# ラン履歴 仕様

## 概要

1ランの進行内容を記録・表示するシステム。スレイザスパイヤのラン履歴に相当する。  
ゲームオーバー画面とタイトル画面の「過去のラン」から閲覧できる。

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ラン情報記録 | ランの進行に合わせてイベントをリアルタイムで記録する |
| 2 | ランリザルト表示 | ゲームオーバー時に当該ランの履歴をUIに表示する |
| 3 | 履歴保存 | ラン終了時にランデータをファイルへ永続保存する |
| 4 | 過去履歴一覧 | タイトル画面から過去のラン一覧を閲覧できる |
| 5 | 過去履歴詳細 | 一覧から選択した過去ランの詳細を表示する |

## 記録内容

| 項目 | 説明 |
|------|------|
| 結果 | ゲームオーバー / エリアクリア数 |
| 到達エリア・ノード | 何エリアのどこまで進んだか |
| 通過したノード一覧 | どのマスをどの順番で通ったか（種別・座標） |
| 最終的な最大体力 | ランク終了時点の最大HP |
| グリッド編成 | 最終バトル時のグリッド配置（タワー種別・位置） |
| 所持タワー | グリッド外の手持ちタワー一覧 |
| 所持強化パーツ | 所持中のレリック一覧 |
| アイテム取得ログ | タワー・強化パーツをどのノードで入手したか |

## データ構造

### RunHistoryData

| フィールド | 型 | 説明 |
|-----------|---|------|
| runId | string | ランの一意ID（開始日時ベース） |
| result | RunResult | GameOver / AreaClear |
| reachedArea | int | 到達エリア数 |
| reachedNodeId | int | 到達した最後のノードID |
| visitedNodes | List\<NodeRecord\> | 通過ノードの記録（順番付き） |
| finalMaxHp | int | ラン終了時の最大HP |
| finalGridLayout | TowerRecord[,] | 最終グリッド配置 |
| finalInventory | List\<TowerRecord\> | 最終手持ちタワー |
| finalRelics | List\<string\> | 最終所持強化パーツIDリスト |
| acquireLog | List\<AcquireRecord\> | アイテム取得ログ |

### NodeRecord

| フィールド | 型 | 説明 |
|-----------|---|------|
| nodeId | int | ノードID |
| nodeType | NodeType | ノード種別 |
| visitOrder | int | 訪問順（1始まり） |

### AcquireRecord

| フィールド | 型 | 説明 |
|-----------|---|------|
| nodeId | int | 取得したノードID |
| nodeType | NodeType | ノード種別 |
| itemType | ItemType | Tower / Relic / Scrap |
| itemId | string | アイテムID |
| source | AcquireSource | Drop / Shop / Event / Reward |

## RunManagerへの追加

| フィールド | 型 | 説明 |
|-----------|---|------|
| `CurrentRunHistory` | RunHistoryData | 進行中のランの記録（都度更新） |

| メソッド | 説明 |
|---------|------|
| `RecordNodeVisit(NodeRecord)` | ノード訪問時に呼ぶ |
| `RecordAcquire(AcquireRecord)` | アイテム取得時に呼ぶ |
| `FinalizeRunHistory(RunResult)` | ラン終了時に呼び、永続保存する |

## 履歴の保存・読み込み

- ラン終了時（ゲームオーバー or 全エリアクリア）に `FinalizeRunHistory` を呼んでファイルへ保存する
- 保存件数の上限: 直近20件（超えた場合は最古を削除）
- SaveManager が管理し、ランデータ（中断セーブ）とは独立したファイルに保存する

## タイトル画面での閲覧

- タイトル画面に「過去のラン」ボタンを追加する（title/spec.md を更新）
- 保存済みランの一覧（日時・到達エリア・結果）を表示する
- 選択すると詳細（ランリザルトと同じレイアウト）を表示する
- 閉じると一覧に戻る

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 途中でアプリが落ちた場合 | 次回起動時に FinalizeRunHistory を呼んで不完全なまま保存する |
| 保存ファイルの読み込み失敗 | 該当ランをスキップして表示しない |

## 未対応ケース
