# ゲームオーバー画面 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ゲームオーバー演出 | ゲームオーバーテキストを表示する |
| 2 | ランリザルト表示 | そのランの履歴（run-history）を画面に表示する |
| 3 | タイトルへ戻る | 画面タップ / クリックでタイトル画面へ遷移する |
| 4 | ランデータのリセット | タイトルへ戻る際にRunManagerのランデータをリセットする |

## 操作方法

| 入力 | アクション |
|------|-----------|
| 画面クリック（どこでも） | タイトル画面へ遷移する |

## ゲームプレイフロー

1. 戦車HPが0になるとバトルから GameOverScene へ遷移する
2. ゲームオーバー演出（テキスト表示）を再生する
3. ランリザルト（run-history）を画面に展開表示する
4. プレイヤーが画面をクリックすると TitleScene へ遷移する
5. 遷移時に RunManager のランデータをリセットする

## 各ゲーム状態の詳細

### ゲームオーバー画面（表示中）

- **表示内容**: ゲームオーバーテキスト、ランリザルト（run-history コンポーネント）
- **プレイヤー操作**: 画面クリックでタイトルへ
- **遷移条件**: 画面クリック
- **遷移先**: TitleScene（フェードあり・RunManagerリセット後）

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| ランデータが不完全（途中から起動など） | 取得できた情報のみ表示し、不足分は「―」で表示する |

## 未対応ケース
