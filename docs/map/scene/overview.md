# MapScene 概要

## 目的・背景

生成済みの `MapGraph` をプレイヤーが閲覧・操作するための Unity シーンと UI。ノードをクリックして各ゲームモードへ遷移するハブ画面として機能する。

## スコープ

**含む:**
- MapScene の Unity シーン構築（Canvas・カメラ設定）
- `MapGraphView` — ノードボタンと接続線の動的描画
- `NodeButtonUI` — ノード状態に応じた見た目制御
- `MapSceneController` — ノード選択受信・シーン遷移ロジック
- MapHUD（エリア番号・スクラップ残高表示）

**含まない:**
- マップグラフの生成ロジック（→ map/generation/）
- ハテナイベントの処理（→ economy/hatena/）
- ショップ・改修・バトルの各シーン実装

## 制約

- `MapGraph` が RunManager に保存済みであること（map/generation/ の前提）
- ノード選択は「到達可能（Reachable）」なノードのみ受け付ける
- ハテナノード選択時のみ別シーンへ遷移せず、MapScene 上にオーバーレイ表示する

## 完了条件

- マップが正しく描画され、到達可能ノードのみクリック可能
- 各ノード種別に応じて正しいシーンへ遷移する
- バトル/ショップ/改修から戻ると、ノードが訪問済み状態になる
