# MapScene 仕様

![マップ画面イメージ](../Images/map_scene_mockup.png)

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ノードグラフ描画 | MapGraph の全ノードをアイコン付きボタンとして動的配置し、接続線を描画する |
| 2 | ノード状態表示 | 各ノードの状態（未訪問・到達可能・現在地・訪問済み）を視覚的に区別する |
| 3 | ノード選択 | 到達可能ノードのみクリック受付。MapSceneController に通知する |
| 4 | シーン遷移 | 選択ノード種別に応じて適切なシーンへ遷移する |
| 5 | MapHUD 表示 | エリア番号・所持スクラップ数を常時表示する |
| 6 | ドラッグスクロール | マップ上でドラッグしてパン移動する |
| 7 | フリーハンド描画 | ペンツールでマップ上に自由な線を描く |
| 8 | 描画消去 | 消しゴムツールで描画した線をフリーハンドで消す |
| 9 | ツール切り替え | 左下のツールバーでペン・消しゴムを切り替え、右クリックで解除する |
| 10 | マップ導入アニメーション | 新エリア到達時にマップ全体を下から上へスクロール表示し、スタートノードにフォーカスする |
| 11 | メニューボタン | 右上のメニューボタンから設定・あきらめる・中断を選択できる |

## ノード状態の表示仕様

| 状態 | 表示 |
|------|------|
| Unvisited（到達不可） | グレーアウト、クリック無効 |
| Reachable（到達可能） | ハイライト（点滅 + 拡縮アニメーション）、クリック有効。マウスオーバー時は最大スケールで静止・点滅停止 |
| Current（現在地） | 戦車アイコンを重ねて表示 |
| Visited（訪問済み） | 暗くなったアイコン + 旗を表示、クリック無効 |

## ノード種別と遷移先

| 種別 | 遷移先 |
|------|--------|
| Battle | PreparationScene（通常バトルフラグ） |
| Boss | PreparationScene（ボスフラグ） |
| Shop | ShopScene |
| Refit | RefitScene |
| Mystery | MapScene 上にハテナイベント UI をオーバーレイ表示 |

## マップ導入アニメーション仕様

- 新エリアのMapSceneに遷移した直後に再生する（既存エリアに戻った場合は再生しない）
- カメラをマップ最下部に配置した状態から最上部まで自動スクロールし、マップ全体をプレイヤーに見せる
- スクロール完了後、スタートノード（現在地）の位置にカメラをフォーカスする
- アニメーション中はノード選択・スクロール操作を受け付けない

## メニューボタン仕様

- 画面右上に常時表示するメニューボタンを配置する
- クリックするとメニューオーバーレイが開く

### メニュー項目

| 項目 | 動作 |
|------|------|
| 設定 | タイトル画面と同じ設定ダイアログをオーバーレイ表示する |
| あきらめる | 確認ダイアログ後、ランを終了してゲームオーバーフロー（ランリザルト表示）へ遷移する |
| 中断 | ランデータを SaveManager に保存し、TitleScene へ遷移する |

### あきらめる確認ダイアログ

- 「ランを終了します。よろしいですか？」「はい」「いいえ」を表示する
- 「はい」: ゲームオーバーと同じフロー（RunHistoryを確定 → GameOverScene）へ遷移する
- 「いいえ」: メニューを閉じてマップ操作に戻る

## マップ進行方向

- ノードは下から上に向かって進む（スタートが最下層、ボスが最上層）
- 一度通過したノードより下の層には戻れない（下方向への遷移は不可）

## ドラッグスクロール仕様

- ペン・消しゴムツールが非アクティブのときのみドラッグでパン移動が有効
- ドラッグ開始位置と現在位置の差分だけマップをオフセットする

## フリーハンド描画仕様

### ツールバー

- 画面左下に「ペン」「消しゴム」のアイコンボタンを配置する
- クリックでそのツールをアクティブにし、カーソルがペン/消しゴムの形に変化する
- 右クリックでアクティブなツールを解除し、カーソルを通常に戻す
- 両方を同時にアクティブにはできない（ペンを選ぶと消しゴムは解除される）

### ペンツール

- ツールアクティブ中にドラッグするとフリーハンドで線を描く（ペイントの鉛筆ツールと同様）
- 描いた線はマップ上のオーバーレイレイヤーに保持される

### 消しゴムツール

- ツールアクティブ中にドラッグすると通過した範囲の描画を消す
- 消しゴムの範囲はペン線の幅より広め（目安: ペン幅の3倍程度）

### 描画の永続性

- 描いた線はマップクリア（ボス撃破）またはゲームオーバーまで保持される
- シーン遷移で MapScene を離れても、戻ってきたときに描画内容が復元される

## 操作方法

| 入力 | アクション |
|------|-----------|
| ノードをクリック（ツール非アクティブ時） | 到達可能なノードを選択して遷移 |
| マップ上でドラッグ（ツール非アクティブ時） | マップをパン移動 |
| Reachableノードにマウスオーバー | 最大スケールで静止・点滅停止 |
| ペンアイコンをクリック | ペンモードに切り替え（カーソルがペン形状に変化） |
| 消しゴムアイコンをクリック | 消しゴムモードに切り替え（カーソルが消しゴム形状に変化） |
| ペン/消しゴムモード中にドラッグ | フリーハンドで線を描く / 消す |
| 右クリック | ペン/消しゴムモードを解除 |
| メニューボタンをクリック | メニューオーバーレイを開く |

## シーン構造

```
MapScene
  ├── [System]
  │     └── MapSceneController (MonoBehaviour)
  │
  ├── [Map]
  │     ├── MapGraphView (MonoBehaviour)
  │     │     ├── EdgeRenderer × n (LineRenderer)
  │     │     └── NodeButton × n (Prefab)
  │     │           ├── NodeButtonUI (MonoBehaviour)
  │     │           ├── Image（アイコン）
  │     │           └── TMP_Text（ラベル）
  │     └── DrawingOverlay（RenderTexture または GL描画レイヤー）
  │
  ├── [HUD]
  │     └── MapHUD
  │           ├── AreaLabel (TMP_Text)
  │           └── ScrapText (TMP_Text)
  │
  ├── [Toolbar]
  │     └── DrawingToolbar（画面左下）
  │           ├── PenButton（ペンアイコン）
  │           └── EraserButton（消しゴムアイコン）
  │
  └── [Menu]
        ├── MenuButton（画面右上）
        └── MenuOverlay（オーバーレイ）
              ├── SettingsButton（設定）
              ├── GiveUpButton（あきらめる）
              └── SuspendButton（中断）
```

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 到達不可ノードをクリック | イベントを無視する（ボタンは非インタラクティブ） |
| RunManager の MapGraph が null | エラーログを出力し、マップ生成を再試行する |
| ペン/消しゴムモード中にノードをクリック | ツール操作を優先し、ノード選択は無視する（誤操作防止） |

## 未対応ケース

- ズーム（拡大・縮小）
- ノード間移動のスプラインアニメーション
- 描画内容のアンドゥ・リドゥ
