# バフ型タワー 概要

## 目的・背景

移動速度・スクラップ効率・全タワーの火力など、戦車のパラメータを底上げするバフ型タワーを導入する。攻撃・防御以外の第三の軸としてビルドの幅を広げる。

## スコープ

### 作るもの

- グリッド配置時にパッシブバフを適用し、除去時に解除する `BuffBehaviour` コンポーネント
- バフ値を集約管理し、戦車・タワーに反映する `BuffManager`（RunManager の一部または別クラス）
- バフ対象: 移動速度・スクラップ取得量・スクラップ回収半径・全タワーCT倍率・最大HP

### 作らないもの

- CTベースの一時バフ（常時パッシブのみ）
- 敵へのデバフ

## 制約

- 同種バフは加算スタックする（上限なし）
- バフ値の再計算はタワーの追加・除去時のみ（毎フレーム計算しない）
- 最大HP バフは設置時に増加、除去時に減少する（除去後HPが最大HPを超えていたら最大HPまでクランプ）

## 完了条件

- バフ型タワーを配置すると対応するパラメータが即時反映される
- 複数積みで効果が加算スタックする
- グリッドから除去するとバフが解除される
