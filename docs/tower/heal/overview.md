# 回復型タワー 概要

## 目的・背景

被ダメを受けながら戦い続けられるよう、CTごとに戦車HPを回復する回復型タワーを導入する。消耗戦での生存力を高めるビルド選択肢として機能させる。

## スコープ

### 作るもの

- CTごとに TankHealth.Heal() を呼ぶ `HealBehaviour` コンポーネント

### 作らないもの

- オーバーヒール（最大HPを超える回復）
- 敵への回復（戦車のみ対象）
- 範囲回復（複数対象への回復）

## 制約

- 回復は TankHealth の最大HPを上限とする
- HP満タンでもCTはリセットする（発動タイミングを一定に保つ）

## 完了条件

- CTごとに指定量のHPが回復する
- 最大HPを超えない
- HP満タン時も正常に動作する
