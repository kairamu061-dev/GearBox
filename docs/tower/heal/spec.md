# 回復型攻撃 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | HP回復 | CTが0になった時点で TankHealth.Heal() を呼び、指定量のHPを回復する |
| 2 | 上限制御 | 回復後のHPが最大HPを超えないよう TankHealth 側でクランプする |
| 3 | CT管理 | HP満タンでもCTをリセットして次の回復サイクルを維持する |

## タワー一覧

| タワー名 | 回復量 | CT(s) | 概要 |
|---------|-------|-------|------|
| 修理機構 | 10 HP/回 | 5.0 | 継続的なHP回復 |

## 動作仕様（HealBehaviour）

1. `TowerBehaviour` がCTを0にすると `IAttackBehaviour.Execute()` を呼ぶ
2. `tankHealth.Heal(data.healAmount)` を呼ぶ
3. CTをリセットする

### TankHealth.Heal() の処理

```
currentHp = min(currentHp + healAmount, maxHp)
```

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| HP満タンで回復が発動 | Heal() 内でクランプされ、HP変化なし。CTはリセットする |
| 複数の回復型タワーを積んだ場合 | 各タワーが独立してCTを管理し、それぞれのタイミングで回復を実行する |

## 未対応ケース

- オーバーヒール（最大HPを超える一時的な上限拡張）
- 敵への回復（味方のみ対象）
- 回復エフェクトのHUD表示
