## ダメージ種別

すべての攻撃タイプは以下のいずれかのダメージ種別を持つ。

| 種別 | 説明 |
|------|------|
| 単体ダメージ | 1体の敵にのみダメージを与える |
| 範囲ダメージ | 指定半径・範囲内の全敵にダメージを与える |

## デバフ

攻撃に付随して敵にデバフを付与できる。デバフの内容はタワーごとに定義する。

| デバフ例 | 説明 |
|---------|------|
| スロー | 移動速度を一定時間低下させる |
| 燃焼 | 一定時間継続ダメージを与える |
| 感電 | 一定時間行動不能または速度低下させる |

> 具体的なデバフ一覧・数値はタワーごとの仕様で定義する。

## 攻撃タイプ別の動作仕様

### 照準型（方向指定）

- マウス位置が射程内：その位置を着弾点として弾を発射する
- マウス位置が射程外：射程距離の境界点を着弾点として発射する
- 弾はPhysics 2D で直進し、着弾点に到達するか敵・壁コライダーに当たると消滅する

### 自動追尾型

- CT が 0 で射程内に敵あり → 発射して CT リセット
- CT が 0 で射程内に敵なし → 発射せず CT もリセットしない
- CT がない状態（未発射 or CT消費済み）で敵が射程内に入る → 即時発射し CT 開始
- 発射後にターゲットが破壊された場合：弾はその破壊地点まで飛行して着弾する。範囲攻撃でなければダメージは発生しない

### 範囲型

- CT到達時に自機を中心とした指定半径内のすべての敵にダメージを与える
- 弾の生成なし（即時ダメージ）
- エフェクトのみ再生する

### 設置型

- CT到達時に設置オブジェクトを生成する
- デフォルトは自機直下に設置。タワーによっては周囲へのばらまき・方向指定発射などの設置方法を持つ
- 設置から当たり判定発生まで遅延があるタワーもある（タワーごとに個別定義）
- 設置オブジェクトは一定時間または敵が触れるまでフィールドに残る
- 設置数の上限はタワーごとに設定（初期: 3個）

### バリア型

- CTごとにシールドHPを展開し、被ダメージをシールドで肩代わりする
- シールドHPが全吸収または持続時間切れで消滅すると同時にCTをリセットし、再展開を待つ
- シールド展開中に再度CTが0になっても二重展開はしない（シールドが残っていればCTのみリセット）

### 迎撃型

- CTが0になった時点で Physics2D.OverlapCircle で射程内の EnemyProjectile レイヤーのオブジェクトを全取得する
- 最も近い敵弾方向へ迎撃弾を発射する
- 迎撃弾は敵にダメージを与えず、EnemyProjectile コライダーに命中した弾を撃墜（即座にプール返却）する
- 電磁迎撃機の迎撃弾は貫通し、通過した複数の敵弾を撃墜できる
- 射程内に敵弾がない場合は CT をリセットせず次フレームで再チェックする（自動追尾型と同様の方針）

### 回復型

- CTごとに戦車HP を healAmount 回復する
- HP上限は最大HPまで（オーバーヒールなし）
- HP満タンでも CT はリセットする

### バフ型

- 設置中はパッシブで効果が発動する（CTなし、常時適用）
- 同じバフ種別のタワーを複数積むと効果が加算スタックする
- バフ値は RunManager が管理し、タワーの追加・除去時に再計算する
- バフ対象: 移動速度・スクラップ取得量・スクラップ回収半径・全タワーCT倍率・最大HP

### ビーム型

- CT到達時にタワー正面（または指定方向）へビームを照射する
- **単体ダメージ**の場合：射程内で最も近い敵1体にのみ当たり、そこでビームの見た目が途切れる
- **範囲ダメージ**の場合：射程内の直線上にいる全敵に当たる
- 詳細仕様は [tower/beam](./beam/spec.md) を参照

## 機能一覧（サマリー）

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | タワー定義 | ScriptableObject でタワーパラメータを管理する |
| 2 | CT管理 | 各タワーインスタンスが独立してCTをカウントする |
| 3 | 弾丸管理 | オブジェクトプールで弾丸のインスタンスを再利用する |
| 4 | 合成 | 2つのタワーを消費して新しいタワーを1つ生成する |
| 5 | レシピ管理 | 合成レシピを既知・未知に分けて管理する |
| 6 | 設計図 | ドロップ品を取得すると未知レシピが既知になる |
| 7 | バリア管理 | CTごとにシールドHPを展開し、被ダメを肩代わりする |
| 8 | 迎撃 | 射程内の敵弾を自動検出して撃墜する |
| 9 | 回復 | CTごとに戦車HPを回復する |
| 10 | バフ管理 | 設置中パッシブで戦車パラメータを強化し、複数積みで加算スタックする |
