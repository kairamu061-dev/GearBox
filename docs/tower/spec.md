## 攻撃タイプ別の動作仕様

### 照準型

- 発射時点のマウスカーソルのワールド座標方向へ弾を発射する
- 弾はPhysics 2D で直進し、敵または壁コライダーに当たると消滅する
- 射程を超えると自動消滅する

### 自動追尾型

- CT到達時に射程内で最もHPが低い（または最も近い）敵をターゲットにする
- ターゲットへ向けて弾を発射する（ホーミングは行わない、発射方向は固定）
- 射程内に敵がいない場合は次のCTまで待機する

### 範囲型

- CT到達時に自機を中心とした指定半径内のすべての敵にダメージを与える
- 弾の生成なし（即時ダメージ）
- エフェクトのみ再生する

### 設置型

- CT到達時に自機の前方（進行方向）に設置オブジェクトを生成する
- 設置オブジェクトは一定時間または敵が触れるまでフィールドに残る
- 設置数の上限はタワーごとに設定（初期: 3個）

## 機能一覧（サマリー）

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | タワー定義 | ScriptableObject でタワーパラメータを管理する |
| 2 | CT管理 | 各タワーインスタンスが独立してCTをカウントする |
| 3 | 弾丸管理 | オブジェクトプールで弾丸のインスタンスを再利用する |
| 4 | 合成 | 2つのタワーを消費して新しいタワーを1つ生成する |
| 5 | レシピ管理 | 合成レシピを既知・未知に分けて管理する |
| 6 | 設計図 | ドロップ品を取得すると未知レシピが既知になる |

## 未対応ケース

- タワーの耐久度（タワーが壊れる仕様）は将来対応
- タワーへのアクセサリ装着は将来対応
- 3種以上のタワーを使う合成は対象外
