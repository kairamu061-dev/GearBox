# ビーム型攻撃 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ビーム照射 | CT が 0 になった時点でタワー正面（または指定方向）へビームを照射する |
| 2 | 単体ダメージ判定 | 単体ダメージの場合、射程内で最も近い敵1体にのみ当たる |
| 3 | 範囲ダメージ判定 | 範囲ダメージの場合、射程内の直線上にいる全敵に当たる |
| 4 | ビーム表示制御 | 単体ダメージ時はビームの見た目が命中した敵の位置で途切れる |

## 動作仕様（BeamAttack）

1. `TowerBehaviour` が CT を 0 にすると `IAttackBehaviour.Execute()` を呼ぶ
2. タワー正面方向へ `Physics2D.RaycastAll` を射程距離まで実行する
3. **単体ダメージ**の場合：
   - ヒットした敵のうち最も近い1体に `TakeDamage()` を呼ぶ
   - ビームエフェクトをタワーから命中点まで表示する（それ以降は描画しない）
4. **範囲ダメージ**の場合：
   - ヒットした全敵に `TakeDamage()` を呼ぶ
   - ビームエフェクトを射程距離全体に表示する

## パラメータ

| パラメータ | 説明 |
|-----------|------|
| ダメージ種別 | 単体 or 範囲（TowerData で定義） |
| 照射方向 | 正面固定 or マウス方向（TowerData で定義） |
| ビーム幅 | エフェクトの幅（ゲームプレイには影響しない） |
| 照射時間 | ビームエフェクトの表示持続時間（秒） |

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 射程内に敵がいない（単体ダメージ） | ダメージなし。ビームは射程端まで表示する |
| 壁・障害物がビーム軌道上にある | 障害物手前で止まる（壁を貫通しない） |

## 未対応ケース

- ビームのチャージ演出（発射前に溜める動作）
- バウンド・反射ビーム
