# 範囲型攻撃 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 範囲内敵検出 | CT が 0 になった時点で Physics2D.OverlapCircle で射程半径内の全敵を取得する |
| 2 | 全敵ダメージ | 取得した全敵に IDamageable.TakeDamage(data.damage) を即時呼ぶ |
| 3 | 範囲エフェクト | ダメージ発生と同時に範囲エフェクト（パーティクル等）を再生する |
| 4 | 空振りスキップ | 射程内に敵がいない場合は CT をリセットし次の CT を待つ |

## 動作仕様（AreaAttack）

1. `TowerBehaviour` が CT を 0 にすると `IAttackBehaviour.Execute()` を呼ぶ
2. `Physics2D.OverlapCircle(transform.position, data.range, enemyLayer)` で全敵を取得する
3. 取得した Collider2D ごとに `IDamageable.TakeDamage(data.damage)` を呼ぶ
4. エフェクト GameObject を有効化する（エフェクトなしでも動作すること）
5. CT をリセットして次の攻撃を待つ（敵がゼロでも CT はリセットする）

## CT リセット方針

敵の有無に関わらず CT を **常にリセット** する。
> **理由:** 範囲型は発動エフェクト（爆発・炎など）のリズムをゲームプレイ感覚として一定に保つ設計のため。自動追尾型（tower/autoaim）は「弾を撃てるときだけ CT を消費する」設計として区別する。

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 射程内に敵がゼロ | エフェクトなしでスキップ、CT はリセット |
| エフェクト GameObject が未設定 | エラーなしでスキップ（エフェクトはオプション） |

## 未対応ケース

- 遮蔽物（障害物）を考慮した視線チェック（現仕様は遮蔽無視）
- ダメージの距離減衰
