# 範囲型攻撃 概要

## 目的・背景

自機周囲の指定半径内に存在するすべての敵に即時ダメージを与える攻撃タイプ。弾丸を使用しないため遮蔽物を無視する。蒸気炎放器などが使用する。

## スコープ

**含む:**
- `AreaAttack` MonoBehaviour（IAttackBehaviour 実装）

**含まない:**
- `Projectile`（この攻撃タイプでは不使用）
- 他の攻撃タイプ

## 制約

- `IAttackBehaviour` インターフェース（tower/base/）に準拠する
- `Physics2D.OverlapCircle` で射程内の全敵を取得し、全員に `IDamageable.TakeDamage()` を呼ぶ
- 弾丸・ObjectPool は使用しない
- 範囲エフェクトを再生する（エフェクトなしでも動作すること）

## 完了条件

- CT ごとに射程内の全敵にダメージが入る
- 遮蔽物（障害物タイルマップ）を無視してダメージが届く
- 射程内に敵がいない場合は無処理でスキップする
