# バリア型タワー 概要

## 目的・背景

攻撃だけでなく防御の選択肢をプレイヤーに与えるため、CTごとにシールドを展開して被ダメを肩代わりするバリア型タワーを導入する。

## スコープ

### 作るもの

- シールドHP を展開・管理する `BarrierBehaviour` コンポーネント
- シールド吸収・消滅・再展開のサイクル処理
- シールド展開中の視覚エフェクト（半透明オーバーレイ）

### 作らないもの

- シールドの方向性（全方位シールドのみ、方向指定なし）
- 複数タワーのシールドHPを合算する処理（各タワーが独立管理）

## 制約

- シールドは戦車全体に適用（タワー単体ではなく戦車を守る）
- 複数のバリア型タワーを積んだ場合、シールドは独立して管理される（合算しない）

## 完了条件

- CTごとにシールドが展開され、被ダメをシールドHPで吸収できる
- シールドHP消滅または時間切れ後にCTリセット・再展開が正しく動作する
- シールド展開中・非展開中でエフェクトが切り替わる
