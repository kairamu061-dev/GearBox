# タワーシステム 概要

## 目的・背景

戦車に積み込む攻撃砲台「タワー」の定義・挙動・合成システムを実装する。タワーの種類・組み合わせ・配置がゲームの戦略的な深みを生み出す中核要素である。

## スコープ

### 作るもの

- タワーの定義（ScriptableObject）: ダメージ・CT・射程・サイズ・攻撃タイプ
- 攻撃タイプ別の攻撃挙動（照準型・自動追尾型・範囲型・設置型）
- 弾丸の生成・移動・当たり判定
- タワー合成システム（2つのタワーから新タワーを生成）
- 合成レシピ管理（既知・未知の区別）

### 作らないもの

- グリッドへの配置UI（battle/preparation で対応）
- タワーのショップ購入・売却処理（economy で対応）
- タワーのアップグレード処理（economy で対応）

## 制約

- タワーのデータはすべて ScriptableObject で定義し、コード変更なしに追加できるようにする
- 弾丸はオブジェクトプールで管理し、大量弾丸によるパフォーマンス低下を防ぐ
- 合成レシピも ScriptableObject で定義する

## 完了条件

- ScriptableObject で定義した初期タワー7種が正しく攻撃する
- 各攻撃タイプが仕様通りに動作する（照準・追尾・範囲・設置）
- 合成画面で2つのタワーを選択して合成できる
- 既知レシピ以外の組み合わせは「未知」として結果が伏せられる

## 画面イメージ

**タワー情報カード（準備フェーズ・ショップ共通）:**
```
+-------------------------+
|  [タワーアイコン]        |
|  蒸気砲                 |
|  DMG: 30   CT: 2.0s    |
|  射程: 8.0  タイプ: 照準 |
|  サイズ: 1×1            |
+-------------------------+
```

**合成画面（改修ノード内）:**
```
+------------------------------------------+
|  [ 合成 ]                                |
|                                          |
|  [蒸気砲 ▼]   +   [機関銃 ▼]   →  [?] |
|                                          |
|  ▶ 合成する                              |
|                                          |
|  既知レシピ一覧:                          |
|  蒸気砲 + 機関銃 → 蒸気ガトリング ✓       |
|  散弾銃 + 炎放器 → 爆炎散弾砲 ✓          |
|  電磁砲 + 機関銃 → ???              ？   |
+------------------------------------------+
```

## サブ項目分割

タワー定義・攻撃挙動と合成システムは独立して実装可能なため分割する。

| サブ項目 | 内容 |
|---------|------|
| `tower/base` | タワー定義・攻撃タイプ別の挙動・弾丸管理 |
| `tower/synthesis` | 合成システム・レシピ管理 |
