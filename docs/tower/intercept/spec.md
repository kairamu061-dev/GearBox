# 迎撃型攻撃 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 敵弾検出 | CTが0になった時点で Physics2D.OverlapCircle で射程内の EnemyProjectile を全取得する |
| 2 | ターゲット選択 | 最も近い敵弾を撃墜対象とする |
| 3 | 迎撃弾発射 | ターゲット方向へ迎撃弾を発射する |
| 4 | 敵弾撃墜 | 迎撃弾が EnemyProjectile コライダーに命中すると敵弾をプール返却（撃墜）する |
| 5 | 貫通撃墜 | 電磁迎撃機の迎撃弾は貫通し、複数の敵弾を連続撃墜できる |
| 6 | 空振りスキップ | 射程内に敵弾がない場合はCTをリセットせず次フレームで再チェックする |

## タワー一覧

| タワー名 | 射程 | CT(s) | 貫通 | 概要 |
|---------|------|-------|------|------|
| 迎撃砲塔 | 6.0 | 0.5 | なし | 近距離高速迎撃 |
| 電磁迎撃機 | 10.0 | 1.0 | あり | 広範囲・貫通で複数弾を撃墜 |

## 動作仕様（InterceptAttack）

1. `TowerBehaviour` がCTを0にすると `IAttackBehaviour.Execute()` を呼ぶ
2. `Physics2D.OverlapCircle(transform.position, data.range, enemyProjectileLayer)` で敵弾を取得する
3. 取得リストが空の場合はCTをリセットせず終了する
4. 最も近い敵弾をターゲットに選ぶ
5. ターゲット方向へ迎撃弾を発射し、CTをリセットする

## 迎撃弾の仕様

| パラメータ | 値 | 説明 |
|-----------|---|------|
| 移動速度 | 15 units/s | 通常弾より高速 |
| 命中レイヤー | EnemyProjectile | 敵弾のみ撃墜（敵本体にはダメージなし） |
| 貫通 | タワーごとに設定 | 電磁迎撃機のみ true |
| 最大射程 | TowerData.range | 超えたら返却 |

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 射程内に敵弾がない | CTをリセットせず次フレームで再チェック |
| 迎撃弾が敵弾と戦車を同時に通過 | EnemyProjectile レイヤーのみ反応（敵本体は無視） |
| 敵弾がプール返却済みのフレームに命中判定 | null チェックでスキップ |

## 未対応ケース

- ホーミング迎撃弾（発射方向固定の直進のみ）
- 迎撃弾が敵本体にダメージを与える仕様
- 迎撃率・撃ち漏らし確率の設定
