# ショップ 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 在庫生成 | ノード訪問時にタワー・アイテムをランダム抽選して在庫を生成する |
| 2 | 在庫表示 | 在庫アイテムをアイコン・名前・価格付きで一覧表示する |
| 3 | 購入 | スクラップを消費してアイテムを購入し、RunManager に反映する |
| 4 | 残高チェック | 残高不足のアイテムは購入ボタンをグレーアウトする |
| 5 | 売り切れ管理 | 購入済みアイテムは sold フラグで再購入不可にする |

## 在庫仕様

| 項目 | 内容 |
|------|------|
| 在庫数 | 3〜5件（ノード訪問時にランダム生成） |
| 在庫種別 | タワー（各種）・設計図（合成レシピ解放）・消耗品（HP回復） |
| 再生成 | 再訪問しても在庫は変化しない（RunManager に保存） |

## アイテム価格

| タワー名 | 基本価格 |
|---------|---------|
| 蒸気砲 | 40⚙ |
| 散弾銃 | 35⚙ |
| 機関銃 | 30⚙ |
| 迫撃砲 | 50⚙ |
| 感知機雷 | 40⚙ |
| 蒸気炎放器 | 35⚙ |
| 電磁砲塔 | 55⚙ |
| 合成タワー | 80〜120⚙ |

| アイテム | 基本価格 | 効果 |
|---------|---------|------|
| 設計図 | 80⚙ | 未知合成レシピ1つを解放 |
| 修理キット | 30⚙ | 戦車HP +30 回復 |
| 大型修理キット | 60⚙ | 戦車HP +70 回復 |

## 画面仕様

- **表示内容**: 在庫アイテム一覧（価格・パラメータ付き）、所持スクラップ数
- **プレイヤー操作**: 「購入」ボタンでアイテムを購入する
- **遷移条件**: 「閉じる」または「出発」ボタン
- **遷移先**: MapScene

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| スクラップ残高不足で購入しようとした | 購入ボタンをグレーアウト（クリック不可） |
| 購入済みアイテムを再度購入しようとした | sold 状態を維持してボタンを無効化 |
