# 改修ノード 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | タワー強化 | スクラップを消費してタワーのパラメータを1段階アップグレードする |
| 2 | HP 修理 | スクラップを消費して戦車 HP を回復する |
| 3 | タワー売却 | タワーをスクラップに変換する |
| 4 | タワー合成 | SynthesisUI を再利用して合成を行う（準備フェーズと同等） |
| 5 | グリッド拡張 | GridExpansionUI を再利用してグリッドを拡張する（準備フェーズと同等） |

## タブ構成

| タブ名 | 内容 |
|--------|------|
| 強化 | TowerUpgradeUI |
| 合成 | SynthesisUI（tower/synthesis 共用） |
| 修理 | RepairUI |
| 拡張 | GridExpansionUI（battle/preparation 共用） |
| 売却 | TowerSellUI |

## タワー強化仕様

- 最大 Lv3 まで強化できる
- Lv3 タワーの強化ボタンはグレーアウト

| タワー種別 | Lv2 効果 | Lv2 コスト | Lv3 効果 | Lv3 コスト |
|-----------|---------|-----------|---------|-----------|
| 蒸気砲 | DMG +15 | 30⚙ | DMG +15, 射程+1 | 50⚙ |
| 機関銃 | CT -0.05s | 25⚙ | CT -0.05s, DMG +3 | 45⚙ |
| その他 | TBD | TBD | TBD | TBD |

## HP 修理仕様

| 修理量 | コスト |
|--------|--------|
| 最大HP の 30% 回復 | 30⚙ |
| 最大HP の 70% 回復 | 60⚙ |
| 最大HP まで全回復 | 100⚙ |

- 戦車 HP が最大の場合、修理ボタンはすべてグレーアウト

## タワー売却仕様

- 売却価格 = 基本価格の 50%
- 強化済みタワーは強化コストの 25% も上乗せ

## 画面仕様

- **表示内容**: タブ形式（5タブ）、所持スクラップ数
- **遷移条件**: 「出発」ボタン
- **遷移先**: MapScene

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| スクラップ残高不足で強化・修理しようとした | 各ボタンをグレーアウト |
| Lv3 タワーをさらに強化しようとした | 強化ボタンをグレーアウト |
| HP 満タンで修理しようとした | 修理ボタンをグレーアウト |
