# ハテナノード 概要

## 目的・背景

ハテナ（?）ノードで発生するランダムイベントシステムを実装する。ポジティブ・ネガティブ・ニュートラルなイベントが混在することで、マップ上のリスクリワード判断に奥行きをもたせる。

## スコープ

### 作るもの

- `HatenaEventData` ScriptableObject（イベント1件のデータ: 種別・フレーバーテキスト・効果パラメータ）
- 7種のイベントアセット作成
- `HatenaEventController`（イベント抽選・処理実行・オーバーレイ UI 表示）

### 作らないもの

- ショップ購入・改修（他サブ項目で対応）
- スクラップ増減の管理（core/RunManager で対応）

## 制約

- イベントは ScriptableObject で定義し、出現率の調整をコード変更なしに行えるようにする
- イベント UI は MapScene 上のオーバーレイとして表示する（専用シーンは作らない）

## 完了条件

- ハテナノードを踏むとランダムにイベントが選ばれ UI が表示される
- 全7種のイベントが正しく処理される
