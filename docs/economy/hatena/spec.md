# ハテナノード 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | イベント抽選 | ハテナノードを踏んだ際に出現率に従ってイベントを1件選ぶ |
| 2 | イベント処理 | 選ばれたイベントの効果を RunManager に反映する |
| 3 | オーバーレイ UI | MapScene 上にイベント内容と選択肢を表示する |

## イベント一覧

| イベント名 | 内容 | 発生率 |
|-----------|------|--------|
| タワー選択 | ランダムタワー3種から1つを選んで獲得する | 25% |
| スクラップ発見 | スクラップ +30〜80 獲得 | 20% |
| 設計図発見 | ランダムな未知レシピを1つ解放する | 10% |
| 廃材の罠 | スクラップ -20〜50（最低 0 止め） | 15% |
| 謎の部品 | 所持タワーのうちランダムな1体を1レベル強化する | 10% |
| 廃兵の遺産 | HP を最大値まで全回復する | 10% |
| 何もない | 何も起きない | 10% |

## 画面仕様

- MapScene 上にオーバーレイ（半透明パネル）として表示する
- 「タワー選択」イベントのみタワー3つのボタンを表示し、選択後に効果を適用する
- それ以外のイベントはフレーバーテキストを表示して「閉じる」ボタンで終了する

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 「廃材の罠」でスクラップが足りない | スクラップを 0 まで減らして止める（マイナスにしない） |
| 「設計図発見」で未知レシピがない | 「何もない」にフォールバックする |
| 「謎の部品」でLv3タワーしか持っていない | 「何もない」にフォールバックする |
