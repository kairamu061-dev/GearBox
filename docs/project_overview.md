# プロジェクト概要

## ゲームコンセプト

スチームパンクな世界観で、廃材から組み上げた戦車に砲台を積み込み、敵の陣営を突破していくローグライクアクションゲーム。スレイザスパイヤ式のマップを進みながら戦車を強化し、ボスを倒してエリアをクリアする。

## コアループ

1. マップ画面でノードを選択する
2. ノード種別に応じたイベントをこなす（バトル・ショップ・改修など）
3. バトルでは準備フェーズでタワーをグリッドに積み込み、戦車を操作して敵を倒す
4. スクラップを回収し、タワーの購入・強化・合成に使う
5. ボスノードを撃破してエリアをクリアし、次のエリアへ進む

## スコープ

### 実装するもの

- ローグライク式マップ（ノード選択・分岐）
- ノード種別: バトル・ハテナ・ショップ・改修・ボス
- バトルシステム（準備フェーズ + 戦闘フェーズ）
- 戦車グリッド（タワー積み込み管理）
- タワーシステム（複数種・攻撃タイプ・CT制）
- タワー合成システム
- 敵システム（複数種・スクラップドロップ）
- スクラップ経済（通貨・ショップ・改修）
- ミニマップ（旗方向表示）
- ゲームオーバー / エリアクリア

### 実装しないもの

- オンライン要素・マルチプレイ
- ステージエディタ
- ゲームパッド対応（初期リリース時点）
- 音声・BGM（アセット準備できたら追加）

## 技術スタック概要

| 技術 | 用途 | 選定理由 |
|------|------|----------|
| Unity 2022 LTS | ゲームエンジン | 2D機能充実・長期サポート |
| Unity Input System | 入力管理 | キーボード・マウス統一管理 |
| DOTween | UI アニメーション | 軽量・直感的な Tween 制御 |
| Unity Physics 2D | 戦車・弾・衝突判定 | 標準物理で十分な精度 |

## 全体アーキテクチャ

```
シーン構成:
  LogoScene           ← 起動直後のロゴ表示（フェードイン→表示→フェードアウト）
  TitleScene          ← タイトル・ゲーム開始
  MapScene            ← ローグライクマップ画面
  PreparationScene    ← バトル前の積み込み準備画面
  BattleScene         ← 戦闘フィールド
  ShopScene           ← ショップ
  RefitScene          ← 改修（強化・合成・修理・拡張・売却）
  GameOverScene       ← ゲームオーバー・リスタート選択

主要 Manager（DontDestroyOnLoad）:
  GameManager         ← ゲーム全体の状態管理・シーン遷移
  RunManager          ← 1ランン分の進行状態（マップ進捗・所持スクラップ・タワー）
  SceneTransitionManager ← シーン間フェード演出の制御
  AudioManager        ← BGM / SE 再生
  SaveManager         ← セーブデータ管理（ランン中の中断復帰）
```

## 制約

- Unity 2022 LTS の標準 2D 機能のみで実装（外部物理ライブラリ不使用）
- ソロ開発のためアセットは既製素材を活用する
- 初期フェーズでは1エリア（5〜8ノード + ボス）を動作するプロトタイプを目標とする

## UI デザイン方針

### コンセプト

スチームパンク世界観に合わせた「錆・鉄・蒸気」を感じるダークブラウン・アンバー系のビジュアル。HUD は最小限にとどめ、戦闘の視認性を優先する。

### カラーパレット

| 用途 | カラーコード | 説明 |
|------|-------------|------|
| プライマリ | `#C87941` | アンバーブラウン。ボタン・強調 |
| セカンダリ | `#4A7C8E` | スチールブルー。アクセント |
| 背景 | `#1A1410` | 深いダークブラウン。フィールド背景 |
| UI テキスト | `#E8D5A3` | 古紙色。HUD・UI文字 |
| 警告 / ダメージ | `#C0392B` | 赤。HP減少・危険状態 |
| スクラップ | `#A8B820` | 黄緑。スクラップ表示 |

### タイポグラフィ

- フォント: `Noto Sans JP`（日本語UI）、機械的なセリフ体（タイトル・スコア装飾）
- HUD: 14〜16px bold
- タイトル: 28〜36px

### コンポーネント方針

- **ボタン**: 金属板風の角張ったデザイン、ホバー時に蒸気エフェクト
- **HUD**: 画面上部（HP・スクラップ）+ 右下（ミニマップ）
- **ダイアログ**: 鉄枠付きのオーバーレイパネル
