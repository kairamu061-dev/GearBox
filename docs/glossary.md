# 用語集

このプロジェクトで使用するドメイン固有の用語を定義する。
実装・設計・仕様書内で用語を使う際はここに定義されたものに統一する。

---

| 用語 | 定義 |
|------|------|
| ラン | ゲーム開始からゲームオーバーまたはエンディングまでの一周のプレイ。ローグライクにおける「1プレイ」 |
| ノード | マップ上に配置された立ち寄り地点。バトル・ハテナ・ショップ・改修・ボスの種別がある |
| エリア | 複数のノードで構成されるマップの1区画。末尾にボスノードが置かれる |
| 戦車 | プレイヤーが操作するユニット。グリッドにタワーを積み込み、フィールドを移動する |
| グリッド | 戦車上に設けられたタワー積み込み用のマス目。初期サイズは 3×3 |
| タワー | グリッドに配置する攻撃砲台。種類によって攻撃タイプ・CT・サイズが異なる |
| CT（クールタイム） | タワーが次の攻撃を行うまでの待機時間（秒） |
| 照準型 | マウスカーソルの方向に弾を発射するタワーの攻撃タイプ |
| 自動追尾型 | 最も近い敵を自動でロックして攻撃するタワーの攻撃タイプ |
| 範囲型 | 自機周囲の一定半径内に即時ダメージを与えるタワーの攻撃タイプ |
| 設置型 | フィールド上にオブジェクト（地雷・バリアなど）を展開するタワーの攻撃タイプ |
| 合成 | 2つのタワーを組み合わせて新しいタワーを生成するシステム |
| スクラップ | 敵を倒したときにドロップするゲーム内通貨。ショップ・改修で使用する |
| 旗 | バトルノードのクリア目的地。旗の範囲内に戦車が入るとクリア |
| 準備フェーズ | バトル開始前にタワーをグリッドへ配置する画面・フェーズ |
| 戦闘フェーズ | WASDで戦車を操作し、敵を倒して旗を目指すフェーズ |
| ミニマップ | 画面隅に表示される俯瞰マップ。旗の方向・自機位置を表示する |
| ボス | エリアの最終ノードに配置される強力な敵。撃破でエリアクリア |
| 設計図 | タワー合成レシピを解放するアイテム。未知の合成が可能になる |
| ハテナノード | ランダムイベントが発生するノード。報酬・罠・選択肢などが出現する |
| 改修ノード | タワー強化・合成・グリッド拡張・HP修理が行えるノード |
