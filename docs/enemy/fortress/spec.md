# 砦 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | フィールド固定配置 | バトル開始時にフィールド上の固定位置に配置される |
| 2 | 接近検知 | プレイヤー戦車が一定範囲内に入ると攻撃・敵生産を開始する |
| 3 | 攻撃 | 接近中のプレイヤーへ弾を発射する |
| 4 | 敵戦車生産 | 一定間隔でフィールドに敵戦車を生産・出撃させる |
| 5 | HP管理 | ダメージを受けてHP0で破壊される |
| 6 | ドロップ | 破壊時にスクラップ・タワー・強化パーツをランダムドロップする |
| 7 | ミニマップ表示 | ミニマップ上に専用アイコンで砦の位置を表示する |

## 砦のパラメータ

| パラメータ | 値 | 説明 |
|-----------|---|------|
| HP | TBD | 砦の体力 |
| 接近検知半径 | TBD | この距離以内に入ると行動開始 |
| 攻撃CT | TBD | 弾を発射する間隔（秒） |
| 敵生産CT | TBD | 敵戦車を生産する間隔（秒） |
| 生産上限 | TBD | 同時に出撃できる配下敵戦車の最大数 |

## 動作仕様

### 待機状態（プレイヤーが検知範囲外）

- 攻撃・敵生産を行わない
- ミニマップには常時表示する

### 活性状態（プレイヤーが検知範囲内）

- プレイヤーに向けて一定CTで弾を発射する
- 一定CTで敵戦車を生産し、プレイヤーへ向かわせる
- プレイヤーが範囲外に離れると待機状態に戻り、CTはリセットしない

## ドロップ仕様

- 破壊時にドロップテーブルから抽選してアイテムをドロップする
- ドロップ品はフィールド上に出現し、プレイヤーが近づくと回収される（他のドロップと同様）
- 回収後はリザルト画面で反映

### ドロップテーブル（暫定）

| 種別 | 内容 | 備考 |
|------|------|------|
| スクラップ | 一定量 | 必ずドロップ |
| タワー | ランダム1種 | 確率ドロップ |
| 強化パーツ | ランダム1種 | 確率ドロップ |

> 具体的なドロップ率・量はバランス調整時に決定する。

## クリア条件との関係

- 砦の破壊はクリア条件ではない
- スルーしてゴール到達でもバトルクリアとなる
- 任意チャレンジとしてドロップ品目当てで攻略する対象

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 砦が複数あり同時に活性化した場合 | それぞれ独立して動作する |
| 生産上限に達した状態で生産CTが来た | 配下数が上限未満になるまで生産をスキップする |

## 未対応ケース

- 砦の種別バリエーション（強化版・ボス砦など）は将来対応
- 砦の修復・復活
