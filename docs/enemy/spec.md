# 敵システム 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | EnemyData 定義 | ScriptableObject で敵のパラメータを定義する |
| 2 | 追跡AI | 戦車を目標として移動する |
| 3 | 攻撃AI | 射程内の戦車に対して攻撃する |
| 4 | HP管理 | ダメージを受けてHPを減少させ、0でデスする |
| 5 | スクラップドロップ | デス時にスクラップオブジェクトをフィールドに生成する |
| 6 | ボス行動 | HPフェーズに応じて行動パターンが変化する |

## AI タイプ別動作仕様

### 追跡型（Chaser）

1. 毎フレーム戦車の方向を計算し、移動ベクトルを更新する
2. 障害物に当たった場合は回り込む（簡易 Obstacle Avoidance）
3. 戦車に接触するとダメージを与え、ノックバックさせる

### 固定砲撃型（Turret）

1. 移動しない（固定位置に配置）
2. 一定間隔で射程内の戦車をロックして弾丸を発射する
3. 射程外に戦車がいる場合は待機する

### 直線突進型（Rusher）

1. バトル開始時に移動方向（直線）を決定する
2. 決定した方向へ一定速度で移動し、障害物または壁で反転する
3. 戦車と接触するとノックバック＋ダメージ

### 固定要塞型（Fortress）

1. 移動しない（固定位置に配置）
2. 射程内の戦車を砲撃する（固定砲撃型と同様）
3. HP が 50% 以下になると周囲に子砲台（スクラップウォーカー×2）をスポーンする

### ボス型（Boss）

| フェーズ | HP閾値 | 行動 |
|---------|--------|------|
| フェーズ1 | 100%〜60% | 低速移動 + 通常砲撃（CT:2.0s） |
| フェーズ2 | 60%〜30% | 中速移動 + 高速砲撃（CT:1.0s） + 子ユニットスポーン |
| フェーズ3 | 30%〜0% | 高速移動 + 全方位弾 + 突進 |

## EnemyData ScriptableObject フィールド

| フィールド | 型 | 説明 |
|-----------|---|------|
| enemyName | string | 表示名 |
| prefab | GameObject | 敵Prefab |
| maxHp | float | 最大HP |
| moveSpeed | float | 移動速度（固定型は0） |
| attackDamage | float | 1回の攻撃ダメージ |
| attackCooldown | float | 攻撃CT（秒） |
| attackRange | float | 攻撃射程（ユニット） |
| aiType | AIType | AI種別 |
| scrapDropMin | int | スクラップドロップ最小値 |
| scrapDropMax | int | スクラップドロップ最大値 |

## スクラップドロップ仕様

- 死亡時に scrapDropMin〜scrapDropMax のランダム値分のスクラップオブジェクトを生成する
- スクラップオブジェクトは死亡位置の周囲にランダム散乱する
- 戦車が回収半径（2.0 ユニット）以内に近づくと自動回収される
- スクラップオブジェクトは60秒後に自動消滅する

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 敵が障害物に挟まれてスタックした | 2秒経過で強制リスポーン（デスポーンして別位置に再出現） |
| ボスのフェーズ移行中に大ダメージを受けた | フェーズ処理を優先し、フェーズ移行後に通常処理を再開 |
