# ロゴシーン 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ロゴシーケンス再生 | ロゴリストを順番に1枚ずつフェードイン→保持→フェードアウトで表示する |
| 2 | ロゴフェードイン | 現在のロゴ画像のAlphaを 0→1 へアニメーションする |
| 3 | ロゴ表示保持 | フェードイン完了後、一定時間ロゴを表示し続ける |
| 4 | ロゴフェードアウト | 保持時間終了後にロゴ画像のAlphaを 1→0 へアニメーションする |
| 5 | 次ロゴへ進行 | フェードアウト完了後、次のロゴがあれば続けて再生する |
| 6 | 自動遷移 | 全ロゴのシーケンス完了後、TitleScene へ即時遷移する |
| 7 | クリックスキップ | 任意のタイミングでクリックすると残りのシーケンスをすべてキャンセルし、即時 TitleScene へ遷移する |

## 操作方法

| 入力 | アクション |
|------|-----------|
| 画面クリック（マウス左ボタン） | 残りのロゴ演出をすべてスキップして TitleScene へ即時遷移 |

## シーンフロー

1. LogoScene が読み込まれる（画面は黒・ロゴ不可視）
2. ロゴリストの先頭から順に以下を繰り返す:
   a. ロゴ画像のAlphaが 0→1 にアニメーションする（1.5秒）
   b. ロゴをそのまま表示し続ける（2.0秒）
   c. ロゴ画像のAlphaが 1→0 にアニメーションする（1.5秒）
3. 全ロゴ再生完了後、`SceneTransitionManager.TransitionImmediate("TitleScene")` を呼んで遷移する

## タイミング（1ロゴあたり）

| フェーズ | 時間 | 備考 |
|---------|------|------|
| フェードイン | 1.5秒 | Alpha 0 → 1 |
| 表示保持 | 2.0秒 | Alpha 1 で静止 |
| フェードアウト | 1.5秒 | Alpha 1 → 0 |
| 合計（スキップなし） | 5.0秒/枚 | ロゴ枚数分繰り返す |

## ロゴリストの管理

- ロゴは `LogoSceneController` にインスペクタ上でリストとして登録する
- 各エントリは「ロゴ画像（Sprite）」「フェードイン時間」「保持時間」「フェードアウト時間」を持つ
- リストが空の場合は即時 TitleScene へ遷移する

## 各ゲーム状態の詳細

### フェードイン中

- **表示内容**: 黒背景、現在のロゴ画像（Alpha 0→1）
- **プレイヤー操作**: クリックで全シーケンスをスキップ
- **遷移条件A**: フェードイン完了
- **遷移先A**: 表示保持フェーズ
- **遷移条件B**: クリック
- **遷移先B**: TitleScene（即時）

### 表示保持中

- **表示内容**: 黒背景、現在のロゴ画像（Alpha 1）
- **プレイヤー操作**: クリックで全シーケンスをスキップ
- **遷移条件A**: 保持時間（2.0秒）経過
- **遷移先A**: フェードアウトフェーズ
- **遷移条件B**: クリック
- **遷移先B**: TitleScene（即時）

### フェードアウト中

- **表示内容**: 黒背景、現在のロゴ画像（Alpha 1→0）
- **プレイヤー操作**: クリックで全シーケンスをスキップ
- **遷移条件A**: フェードアウト完了・次のロゴあり
- **遷移先A**: 次のロゴのフェードインフェーズ
- **遷移条件B**: フェードアウト完了・次のロゴなし
- **遷移先B**: TitleScene（即時）
- **遷移条件C**: クリック
- **遷移先C**: TitleScene（即時）

## シーン遷移の扱い

LogoScene → TitleScene はフェードなし（`TransitionImmediate`）で遷移する。ロゴ自体がフェードアウトで終わるため、追加フェードを行うと二重になるため。

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| フェードイン開始直後（1フレーム目）にクリック | スキップ処理を実行して TitleScene へ遷移 |
| ロゴリストが空 | 即時 TitleScene へ遷移（エラーなし） |
| ロゴ画像アセットが未設定のエントリがある | そのエントリをスキップしてログ出力、次のロゴへ進む |

## 未対応ケース

- キーボード・コントローラーによるスキップ（将来対応）
- ロゴごとに個別のスキップ（クリックは常に全シーケンスをスキップ）
