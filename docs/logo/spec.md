# ロゴシーン 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ロゴフェードイン | シーン開始と同時にロゴ画像のAlphaを 0→1 へアニメーションする |
| 2 | ロゴ表示保持 | フェードイン完了後、一定時間ロゴを表示し続ける |
| 3 | ロゴフェードアウト | 保持時間終了後にロゴ画像のAlphaを 1→0 へアニメーションする |
| 4 | 自動遷移 | フェードアウト完了後、TitleScene へ即時遷移する |
| 5 | クリックスキップ | 任意のタイミングでクリックするとフェードアウトをキャンセルし、即時 TitleScene へ遷移する |

## 操作方法

| 入力 | アクション |
|------|-----------|
| 画面クリック（マウス左ボタン） | ロゴ演出をスキップして TitleScene へ即時遷移 |

## シーンフロー

1. LogoScene が読み込まれる（画面は黒・ロゴ不可視）
2. ロゴ画像のAlphaが 0→1 にアニメーションする（1.5秒）
3. ロゴをそのまま表示し続ける（2.0秒）
4. ロゴ画像のAlphaが 1→0 にアニメーションする（1.5秒）
5. `SceneTransitionManager.TransitionImmediate("TitleScene")` を呼んで遷移する

## タイミング

| フェーズ | 時間 | 備考 |
|---------|------|------|
| フェードイン | 1.5秒 | Alpha 0 → 1 |
| 表示保持 | 2.0秒 | Alpha 1 で静止 |
| フェードアウト | 1.5秒 | Alpha 1 → 0 |
| 合計（スキップなし） | 5.0秒 | |

## 各ゲーム状態の詳細

### フェードイン中

- **表示内容**: 黒背景、ロゴ画像（Alpha 0→1）
- **プレイヤー操作**: クリックでスキップ可能
- **遷移条件A**: フェードイン完了
- **遷移先A**: 表示保持フェーズ
- **遷移条件B**: クリック
- **遷移先B**: TitleScene（即時）

### 表示保持中

- **表示内容**: 黒背景、ロゴ画像（Alpha 1）
- **プレイヤー操作**: クリックでスキップ可能
- **遷移条件A**: 保持時間（2.0秒）経過
- **遷移先A**: フェードアウトフェーズ
- **遷移条件B**: クリック
- **遷移先B**: TitleScene（即時）

### フェードアウト中

- **表示内容**: 黒背景、ロゴ画像（Alpha 1→0）
- **プレイヤー操作**: クリックでスキップ可能
- **遷移条件A**: フェードアウト完了
- **遷移先A**: TitleScene（即時）
- **遷移条件B**: クリック
- **遷移先B**: TitleScene（即時）

## シーン遷移の扱い

LogoScene → TitleScene はフェードなし（`TransitionImmediate`）で遷移する。ロゴ自体がフェードアウトで終わるため、追加フェードを行うと二重になるため。

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| フェードイン開始直後（1フレーム目）にクリック | スキップ処理を実行して TitleScene へ遷移 |
| ロゴ画像アセットが未設定 | エラーをログ出力し、そのまま TitleScene へ遷移（ロゴなしでスキップ扱い） |

## 未対応ケース

- パブリッシャーロゴ・エンジンロゴの連続表示
- キーボード・コントローラーによるスキップ（将来対応）
