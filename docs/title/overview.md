# タイトルシーン 概要

## 目的・背景

ゲームの入口として、新規ランの開始・中断ランの再開・設定・終了をプレイヤーに提示する。中断データは1スロットのみ保持し、シンプルな選択を維持する。

## スコープ

### 作るもの

- TitleScene（ゲームロゴ・ボタン群の表示）
- 中断データの有無でボタン構成を切り替えるロジック
- 「あたらしくはじめる」実行時の確認ダイアログ
- 設定パネル（オーバーレイ。詳細は settings/spec.md で定義）

### 作らないもの

- 複数セーブスロットの選択UI
- キャラクター・クラス選択（本作には存在しない）
- クレジット画面（将来対応）

## 制約

- 中断データは SaveManager が管理する1スロットのみ
- TitleScene への遷移は LogoScene からはフェードなし（FadeIn のみ実行）、それ以外（ゲームオーバー等）はフェードあり
- 設定パネルはシーン遷移なしでオーバーレイ表示する

## 完了条件

- 中断データの有無でボタン構成が正しく切り替わる
- 各ボタンが正しい遷移先へ進む
- 「あたらしくはじめる」で確認ダイアログが表示され、承認後に中断データが削除される
- 「おわる」でアプリが終了する

## 画面イメージ

```
+-------------------------------+
|  GearBox（ゲームロゴ）        |
|                               |
|  [ はじめる          ]        |  ← 中断データなし
|  [ せってい          ]        |
|  [ おわる            ]        |
+-------------------------------+

+-------------------------------+
|  GearBox（ゲームロゴ）        |
|                               |
|  [ つづきから        ]        |  ← 中断データあり
|  [ あたらしくはじめる ]        |
|  [ せってい          ]        |
|  [ おわる            ]        |
+-------------------------------+
```
