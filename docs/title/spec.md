# タイトルシーン 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | ボタン構成切り替え | 中断データの有無で表示ボタンを切り替える |
| 2 | はじめる | 中断データなし時に表示。新規ランを開始してMapSceneへ遷移する |
| 3 | つづきから | 中断データあり時に表示。中断地点のシーンへ遷移する |
| 4 | あたらしくはじめる | 中断データあり時に表示。確認ダイアログ後、中断データを削除して新規ランを開始する |
| 5 | せってい | 設定パネルをオーバーレイ表示する |
| 6 | おわる | アプリを終了する |
| 7 | フェードイン | TitleScene ロード時に SceneTransitionManager.FadeIn() を実行して画面を表示する |

## 操作方法

| 入力 | アクション |
|------|-----------|
| 「はじめる」クリック | 新規ランを開始して MapScene へ遷移 |
| 「つづきから」クリック | 中断地点のシーンへ遷移 |
| 「あたらしくはじめる」クリック | 確認ダイアログを表示 |
| 「せってい」クリック | 設定パネルをオーバーレイ表示 |
| 「おわる」クリック | アプリを終了 |

## シーンフロー

1. TitleScene がロードされる（画面は黒）
2. `SceneTransitionManager.FadeIn()` を実行して画面をフェードイン表示する
3. SaveManager に中断データがあるかチェックする
4. ボタン構成を決定して表示する（下記ボタン状態参照）
5. プレイヤーのボタン操作を待つ

## 各ゲーム状態の詳細

### 通常待機（中断データなし）

- **表示内容**: ゲームロゴ、「はじめる」「せってい」「おわる」ボタン
- **プレイヤー操作**: 各ボタンをクリック
- **遷移条件**: 「はじめる」クリック
- **遷移先**: MapScene（フェードあり・新規ランを RunManager で初期化してから遷移）

### 通常待機（中断データあり）

- **表示内容**: ゲームロゴ、「つづきから」「あたらしくはじめる」「せってい」「おわる」ボタン
- **プレイヤー操作**: 各ボタンをクリック
- **遷移条件A**: 「つづきから」クリック
- **遷移先A**: 中断地点のシーン（フェードあり・SaveManager からランデータを復元して遷移）
- **遷移条件B**: 「あたらしくはじめる」クリック
- **遷移先B**: 確認ダイアログ表示

### 確認ダイアログ（あたらしくはじめる）

- **表示内容**: 「現在の中断データは削除されます。よろしいですか？」「はい」「いいえ」ボタン
- **プレイヤー操作**: 「はい」または「いいえ」をクリック
- **遷移条件A**: 「はい」クリック
- **遷移先A**: SaveManager.DeleteSave() 実行 → MapScene（フェードあり・新規ランで初期化）
- **遷移条件B**: 「いいえ」クリック
- **遷移先B**: 通常待機（中断データあり）に戻る

### 設定パネル表示中

- **表示内容**: 設定パネル（オーバーレイ）、背景のタイトル画面は暗くなる
- **プレイヤー操作**: 設定を変更、「閉じる」で戻る（詳細は settings/spec.md）
- **遷移条件**: 「閉じる」クリック or パネル外クリック
- **遷移先**: 通常待機状態へ戻る

## ボタン構成まとめ

| 中断データ | 表示ボタン |
|-----------|-----------|
| なし | はじめる／せってい／おわる |
| あり | つづきから／あたらしくはじめる／せってい／おわる |

## 遷移時のフェード方針

| 遷移元 | 遷移先 | フェード |
|--------|--------|---------|
| LogoScene | TitleScene | フェードなし（FadeIn のみ） |
| TitleScene | MapScene | フェードあり |
| TitleScene | 中断地点シーン | フェードあり |
| GameOverScene | TitleScene | フェードあり |

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| SaveManager の読み込みに失敗した | 中断データなし扱いで「はじめる」構成を表示し、エラーをログ出力 |
| 「つづきから」で復元先シーンが不明 | MapScene へフォールバックして遷移 |
| 「おわる」をクリック | `Application.Quit()` を呼ぶ（エディタ上では停止） |

## 未対応ケース

- 複数セーブスロット
- クレジット表示ボタン（将来対応）
- キーボード・コントローラーでのボタン操作（将来対応）
