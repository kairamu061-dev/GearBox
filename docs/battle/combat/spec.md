# 戦闘フェーズ 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 戦車移動 | WASD入力に応じてRigidbody2Dで戦車を移動する |
| 2 | 照準制御 | マウスカーソルのワールド座標から照準方向ベクトルを算出する |
| 3 | タワー自動攻撃 | 各タワーがCTを独立管理し、CT到達で自動攻撃する |
| 4 | 弾丸発射 | 照準型タワーはマウス方向へ弾を発射する |
| 5 | ドロップ品回収 | 戦車の回収コライダーに触れたドロップ品をバトル内で仮保持する |
| 6 | 旗到達判定 | 旗のトリガーに戦車が入るとクリア処理を実行する |
| 7 | ボス撃破判定 | ボスHP0でクリア処理を実行する（ボスノードのみ） |
| 8 | 戦車HP管理 | ダメージを受けるとHP減少、0以下でゲームオーバー |
| 9 | HUD更新 | HPをリアルタイムで更新表示する（スクラップはリザルト画面で反映） |
| 10 | ミニマップ | 自機・敵・砦・旗の位置と旗への矢印を表示する |
| 11 | フィールド生成 | バトル開始時に障害物・敵・砦・旗をフィールドに配置する |

## 操作方法

| 入力 | アクション |
|------|-----------|
| W | 上方向へ移動 |
| S | 下方向へ移動 |
| A | 左方向へ移動 |
| D | 右方向へ移動 |
| マウス移動 | 照準型タワーの発射方向を更新 |

## ゲームプレイフロー

1. PreprationScene から遷移し、BattleScene が開始する
2. RunManager の配置データをもとに戦車グリッドにタワーをインスタンス化する
3. フィールドに障害物・敵・旗が配置される
4. バトルが開始し、プレイヤーはWASDで戦車を操縦する
5. 各タワーがCTに従って自動攻撃を行う
6. 敵・砦を倒すとドロップ品が出現し、戦車が近づくとバトル内で仮保持される
7. クリア条件達成 or ゲームオーバーになるまで続ける
8. クリア条件達成後、クリア演出を経てリザルト画面へ遷移する

## 各ゲーム状態の詳細

### バトル中（通常）

- **表示内容**: フィールド・戦車・敵・旗・HUD・ミニマップ
- **プレイヤー操作**: WASD移動、マウス照準
- **遷移条件A**: 旗のトリガー範囲に戦車が入る
- **遷移先A**: バトルクリア演出 → MapScene
- **遷移条件B**: 戦車HPが0
- **遷移先B**: ゲームオーバー演出 → GameOverScene

### バトルクリア演出

- **表示内容**: クリアテキスト
- **プレイヤー操作**: なし（自動でリザルト画面へ遷移）
- **遷移条件**: 演出完了（約2秒）
- **遷移先**: ResultScene（リザルト画面）

### ゲームオーバー演出

- **表示内容**: ゲームオーバーテキスト
- **プレイヤー操作**: なし（自動で GameOverScene へ遷移）
- **遷移条件**: 演出完了（約2秒）
- **遷移先**: GameOverScene

## フィールド仕様

| 項目 | 内容 |
|------|------|
| フィールドサイズ | 64×64 ユニット（暫定・調整可） |
| 視点 | トップダウン2D、カメラは戦車追従 |
| 外周 | 侵入不可の壁コライダーで囲む |

### 壁の種類

| 種別 | 説明 | HP | 破壊時 |
|------|------|----|--------|
| 壊せない壁 | 永続する障害物。弾・戦車・敵を通さない | — | — |
| 壊せる壁 | HPを持つ障害物。攻撃で破壊できる | 暫定: 60 | 消滅のみ（ドロップなし） |

### フィールド生成

- バトル開始時に壊せない壁・壊せる壁・敵・砦・旗をランダム配置する
- 旗とスタート地点の間には必ず通路が確保される（孤立チェック）

## 戦車パラメータ

| パラメータ | 初期値 | 説明 |
|-----------|--------|------|
| 最大HP | 100 | 戦車の最大体力 |
| 移動速度 | 5.0 | Rigidbody2D の速度（ユニット/秒） |
| 回収半径 | 2.0 | スクラップ自動回収の距離（ユニット） |
| 衝突判定 | CircleCollider2D | 障害物・敵弾との衝突 |

## 勝利 / 敗北 / 終了条件

| 種別 | 条件 | 結果 |
|------|------|------|
| クリア（通常） | 旗のトリガーに戦車が入る | ResultScene へ遷移 |
| クリア（ボス） | ボスのHPが0 | ResultScene へ遷移（エリアクリア） |
| ゲームオーバー | 戦車HPが0以下 | GameOverScene へ遷移 |

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 戦車がフィールド境界に到達 | 壁コライダーで押し返す |
| 敵が全滅したが旗未到達 | 旗へ向かうことでクリア可（敵全滅はクリア条件でない） |
| タワーが0個の状態でバトル開始 | 攻撃なし状態で戦闘継続（プレイヤーの責任） |

## 未対応ケース

- ポーズ・リスタート機能
- バトル中のタワー配置変更
- 複数の旗・分岐目的地
