# 戦闘フェーズ 概要

## 目的・背景

プレイヤーが戦車を操作してフィールドを探索し、敵を倒して旗を目指す戦闘フェーズを実装する。タワーディフェンスとアクションゲームを融合させたコアゲームプレイであり、プレイヤーの操縦技術とタワー配置の相乗効果でバトルを攻略する。

## スコープ

### 作るもの

- WASD による戦車移動（Rigidbody2D）
- マウスカーソルによる照準方向の計算・更新
- 配置済みタワーの自動攻撃ループ（CT管理）
- スクラップの自動回収（接触トリガー）
- 旗の到達判定（CircleCollider2D トリガー）
- ボス撃破判定（ボスHP0）
- 戦車HP管理・ゲームオーバー判定
- HUD（HPバー・スクラップ数・ミニマップ）
- バトルフィールド生成（障害物・敵・旗の配置）

### 作らないもの

- タワーの実装（tower 機能エリアで対応）
- 敵AIの実装（enemy 機能エリアで対応）
- ポーズメニュー（将来対応）

## 制約

- 戦車移動は Rigidbody2D で実装し、障害物と衝突する
- カメラは戦車を常に追従する（Cinemachine または手動追従）
- 照準計算は Camera.main.ScreenToWorldPoint を使用する
- フィールドは Tilemap で構成する

## 完了条件

- WASD で戦車が移動し、障害物に当たって止まる
- マウス方向に照準型タワーが向く
- 配置タワーがCT毎に攻撃する（敵がいなくても発射する）
- 旗に触れるとクリア演出が出て MapScene へ戻る
- 戦車HPが0でゲームオーバー画面へ遷移する
