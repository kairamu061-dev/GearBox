# バトルシステム 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 準備フェーズ | バトル前にタワーをグリッドへ配置する |
| 2 | 戦車移動 | WASDキーで戦車を上下左右に移動する |
| 3 | 照準制御 | マウスカーソルの方向を砲台の照準として使用する |
| 4 | タワー自動攻撃 | 配置タワーがCT毎に自動攻撃を行う |
| 5 | スクラップ自動回収 | 戦車が近くのスクラップに触れると自動で回収する |
| 6 | 旗到達判定 | 旗のトリガー範囲に戦車が入るとバトルクリア |
| 7 | ボス撃破判定 | ボスノードで敵ボスのHPが0になるとバトルクリア |
| 8 | ゲームオーバー判定 | 戦車HPが0になるとゲームオーバー |
| 9 | ミニマップ | 旗の方向・自機位置・敵位置を常時表示する |

## 操作方法

| 入力 | アクション |
|------|-----------|
| W | 前進（上方向） |
| S | 後退（下方向） |
| A | 左移動 |
| D | 右移動 |
| マウス移動 | 照準型タワーの発射方向を指定 |

## ゲームプレイフロー

1. ノード選択後、PreparationScene（準備フェーズ）へ遷移する
2. プレイヤーが所持タワーを戦車グリッドへ配置する
3. 「出撃」ボタンを押すと BattleScene（戦闘フェーズ）へ遷移する
4. バトルフィールドに戦車・敵・旗が配置される
5. プレイヤーがWASDで戦車を操縦し、タワーが自動攻撃する
6. 敵を倒すとスクラップがドロップし、戦車が近づくと自動回収する
7. 旗の範囲に入る（通常バトル）またはボスを倒す（ボスノード）とバトルクリア
8. MapScene に戻り、ノードを訪問済みにして次の選択を待つ

## 各ゲーム状態の詳細

### 準備フェーズ（PreparationScene）

- **表示内容**: 所持タワー一覧、戦車グリッド、タワーのパラメータプレビュー、出撃ボタン
- **プレイヤー操作**: タワーをドラッグ＆ドロップでグリッドへ配置・除去・並び替え
- **遷移条件**: 「出撃」ボタンを押す（タワー未配置でも出撃可）
- **遷移先**: BattleScene

### 戦闘フェーズ（BattleScene）

- **表示内容**: フィールド・戦車・敵・旗・HUD（HP・スクラップ）・ミニマップ
- **プレイヤー操作**: WASD移動、マウス照準
- **遷移条件A（クリア）**: 旗到達 or ボス撃破
- **遷移先A**: MapScene（訪問済み更新）
- **遷移条件B（失敗）**: 戦車HP0
- **遷移先B**: GameOverScene

## HUD 仕様

| 要素 | 位置 | 内容 |
|------|------|------|
| HPバー | 画面上部左 | 現在HP / 最大HP をバーで表示 |
| スクラップ数 | 画面上部右 | 現在の所持スクラップ数 |
| ミニマップ | 画面右下 | 自機・敵・旗の位置、旗への方向矢印 |

## バトルフィールド仕様

| 項目 | 内容 |
|------|------|
| フィールドサイズ | 初期: 64×64 ユニット（調整可） |
| 視点 | トップダウン2D、カメラは戦車追従 |
| 障害物 | 岩・廃材・蒸気管などをランダム配置 |
| 旗 | フィールド上の固定位置に1つ配置。ミニマップと矢印で常時案内 |
| 敵の初期配置 | フィールド上にランダム配置（旗周辺に集中する傾向） |

## 勝利 / 敗北 / 終了条件

| 種別 | 条件 | 結果 |
|------|------|------|
| バトルクリア（通常） | 旗の範囲内に戦車が侵入 | MapScene へ遷移 |
| バトルクリア（ボス） | ボス敵のHPが0 | MapScene へ遷移（エリアクリア処理） |
| ゲームオーバー | 戦車HPが0以下 | GameOverScene へ遷移、ランリセット |

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| タワーを1つも積まずに出撃 | 許可する（ノータワーで戦闘を強いられる） |
| フィールドから戦車が出る | 不可視の壁で弾き返す（Physics 2D の壁コライダー） |

## 未対応ケース

- バトル中のポーズメニュー（将来対応）
- バトルリプレイ・オートプレイ機能
- 複数旗・複数目的地

## サブ項目分割

仕様評価の結果、以下の2つは独立して実装可能なため分割する。

| サブ項目 | 内容 |
|---------|------|
| `battle/preparation` | 準備フェーズ（グリッドUI・タワー配置） |
| `battle/combat` | 戦闘フェーズ（戦車操作・攻撃・判定） |
