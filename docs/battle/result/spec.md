# リザルト画面 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 基本報酬表示 | バトルクリアによる固定報酬（スクラップ・タワー等）を表示する |
| 2 | ドロップ品表示 | バトル中に回収したアイテム（スクラップ・タワー・強化パーツ等）を一覧表示する |
| 3 | アイテム選択 | プレイヤーが受け取るアイテムを選択する |
| 4 | 報酬確定 | 選択したアイテムをRunManagerへ反映してMapSceneへ遷移する |

## 操作方法

| 入力 | アクション |
|------|-----------|
| アイテムをクリック | 受け取るアイテムを選択 / 解除する |
| 「受け取る」ボタンをクリック | 選択アイテムを確定してMapSceneへ遷移する |

## ゲームプレイフロー

1. バトルクリア条件達成後、クリア演出を経てリザルト画面へ遷移する
2. 基本報酬（スクラップ固定量・タワー等）を上段に表示する
3. バトル中に回収したドロップ品を下段に一覧表示する
4. プレイヤーが受け取るアイテムを選択する
5. 「受け取る」ボタンを押すと選択アイテムがRunManagerへ反映され、MapSceneへ遷移する

## 各ゲーム状態の詳細

### リザルト画面（待機中）

- **表示内容**: 基本報酬一覧、ドロップ品一覧（選択UI付き）、所持スクラップ数、「受け取る」ボタン
- **プレイヤー操作**: アイテムの選択 / 解除、「受け取る」ボタン
- **遷移条件**: 「受け取る」ボタンをクリック
- **遷移先**: MapScene（フェードあり）

## 基本報酬

| 報酬種別 | 反映方式 | 備考 |
|---------|---------|------|
| スクラップ | 自動反映（選択不要） | 量はノード・難易度に応じて変動 |
| タワー | 選択式 | 抽選で0〜1個 |

## ドロップ品の扱い

- バトル中に回収したドロップ品はリザルト画面まで「仮保持」状態とし、RunManagerへは反映しない
- リザルト画面で選択・確定した分だけRunManagerへ反映する
- 選択しなかったアイテムは破棄される

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| ドロップ品が0件 | ドロップ品欄に「なし」と表示する |
| 「受け取る」を押さずにアプリを終了 | 次回起動時はリザルト画面から再開し、選択を求める |
