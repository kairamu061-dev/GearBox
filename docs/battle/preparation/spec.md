# 準備フェーズ 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | グリッド表示 | 戦車グリッドを動的サイズで表示する |
| 2 | タワー一覧表示 | 所持タワーをスクロール可能なリストで表示する |
| 3 | タワー選択 | 一覧のタワーをクリックするとカーソルにゴーストが追従する |
| 4 | タワー仮配置 | ゴースト状態でグリッド上に移動するとマスにスナップして半透明表示する |
| 5 | タワー配置確定 | 仮配置中にクリックするとタワーを配置する |
| 6 | タワー除去 | 配置済みタワーをクリックして選択し、グリッド外でクリックすると一覧へ戻す |
| 7 | 占有判定 | タワーのサイズ形状に応じてマスの占有を管理し重複を禁止する |
| 8 | パラメータプレビュー | タワーにホバーするとダメージ・CT・射程などを表示する |
| 9 | 合成プレビュー | タワー選択中に合成対応タワーへホバーすると合成結果を半透明で表示する |
| 10 | 合成実行 | 合成プレビュー中にクリックすると合成を実行する |
| 11 | 選択キャンセル | 一覧・グリッド外クリックまたは右クリックでタワー選択を解除する |
| 12 | タワー合成（タブ） | 合成タブでタワー2つを素材に合成する。グリッド配置済みタワーも手持ちタワーも素材にできる |
| 13 | 出撃 | 「出撃」ボタンで配置データを確定し BattleScene へ遷移する |

## 操作方法

| 入力 | アクション |
|------|-----------|
| 一覧のタワーをクリック | タワーを選択（カーソルにゴーストが追従） |
| 選択中にグリッド上へ移動 | 有効マスにスナップして半透明プレビュー表示 |
| グリッド上の有効マスでクリック | タワーを配置確定 |
| 選択中に合成対応タワーへホバー | 合成結果を半透明でオーバーレイ表示 |
| 合成プレビュー中にクリック | 合成を実行 |
| 一覧・グリッド外クリック or 右クリック | タワー選択を解除 |
| タワーにホバー（未選択時） | パラメータプレビューを表示する |
| 「合成」タブを選択 | 合成UIに切り替える |
| 素材A・B を選択 → 「合成する」ボタン | タワーを合成する（配置済み・手持ち問わず選択可） |
| 「出撃」ボタンをクリック | バトル開始 |

## ゲームプレイフロー

1. MapScene でバトルノードを選択すると PreparationScene へ遷移する
2. 「配置」タブが初期表示される。グリッドは前回バトル終了時の配置状態から開始する（初回は空グリッド）
3. プレイヤーは必要に応じてタブを切り替えて以下の操作を自由に行う
   - 配置タブ: タワーをグリッドへ配置・除去・移動する
   - 合成タブ: グリッド配置済みまたは手持ちのタワー2つを素材に合成する
4. 「出撃」ボタンを押すと配置データが RunManager に保存され BattleScene へ遷移する

## 各ゲーム状態の詳細

### 配置タブ / 待機中

- **表示内容**: 所持タワー一覧・戦車グリッド
- **プレイヤー操作**: タワーをクリックして選択、ホバーでプレビュー表示、タブ切り替え
- **遷移条件**: タワーをクリック
- **遷移先**: タワー選択中状態

### 配置タブ / タワー選択中

- **表示内容**: 選択タワーのゴーストがカーソルに追従、グリッド上では有効マスにスナップして半透明表示（有効:緑/無効:赤）
- **プレイヤー操作**: グリッド有効マスでクリック→配置確定、合成対応タワーへホバー→合成プレビュー、右クリック or 一覧・グリッド外クリック→キャンセル
- **遷移条件A**: グリッド有効マスでクリック
- **遷移先A**: 待機中状態（配置確定）
- **遷移条件B**: 合成対応タワーへホバー
- **遷移先B**: 合成プレビュー中状態
- **遷移条件C**: 右クリック or 一覧・グリッド外クリック
- **遷移先C**: 待機中状態（キャンセル）

### 配置タブ / 合成プレビュー中

- **表示内容**: 合成対象タワーに合成結果の半透明ゴーストをオーバーレイ表示
- **プレイヤー操作**: クリックで合成実行、ホバーを外すとプレビュー解除
- **遷移条件A**: クリック
- **遷移先A**: 待機中状態（合成実行・素材2つ消費・結果タワー手持ちへ追加）
- **遷移条件B**: ホバーを外す
- **遷移先B**: タワー選択中状態

### 合成タブ

- **表示内容**: 素材A・B の選択UI（グリッド配置済み・手持ち両方から選択可）、結果プレビュー（既知なら表示/未知なら???）、「合成する」ボタン、既知レシピ一覧
- **プレイヤー操作**: 素材を選択して「合成する」を押す
- **遷移条件**: タブ切り替え or 「出撃」
- **遷移先**: 他タブ or BattleScene

## グリッド仕様

| 項目 | 内容 |
|------|------|
| 初期サイズ | 3×3 マス |
| マスのサイズ | 64×64 px（UIスペース上） |
| 拡張単位 | 行または列を1つずつ追加（改修ノード専門・準備フェーズでは拡張不可） |
| 占有管理 | 各マスに TowerInstance を格納、null = 空き |

## タワーのサイズ形状仕様

| 表記 | 説明 | 例 |
|------|------|-----|
| 1×1 | 1マス | 全初期タワー |
| 1×2 | 横2マス | 将来の拡張タワー |
| 2×2 | 2×2マス | 将来の大型タワー |
| L字 | カスタム形状 | ScriptableObject で定義 |

## 勝利 / 敗北 / 終了条件

| 種別 | 条件 | 結果 |
|------|------|------|
| 遷移 | 「出撃」ボタンを押す | BattleScene へ遷移 |

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| マスが埋まっている場所へドロップ | ドロップをキャンセルし、タワーを元の位置へ戻す |
| タワーが所持枠からあふれている | スクロールで表示（一覧はスクロール対応） |
| グリッドにタワーが1つもない状態で出撃 | 警告ダイアログを表示せず許可（戦略的選択として有効） |
| 合成後にグリッド配置済みの素材タワーが消費された | グリッドから自動的に除去し、空きマスにする |
| 合成後にグリッド配置済みの素材タワーが使われた | グリッドから自動的に除去し、空きマスにする |

## 未対応ケース

- グリッド上でのタワーの回転
- タワーの重ね置き・スタック
- プリセット保存（お気に入り配置の記憶）
