# 準備フェーズ 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | グリッド表示 | 戦車グリッドを動的サイズで表示する |
| 2 | タワー一覧表示 | 所持タワーをスクロール可能なリストで表示する |
| 3 | タワー配置 | タワーをドラッグ＆ドロップでグリッドのマスへ配置する |
| 4 | タワー除去 | グリッドからタワーをドラッグして一覧へ戻す |
| 5 | 占有判定 | タワーのサイズ形状に応じてマスの占有を管理し重複を禁止する |
| 6 | パラメータプレビュー | タワーにホバーするとダメージ・CT・射程などを表示する |
| 7 | タワー合成 | 合成タブで所持タワー2つを選んで合成を実行する（tower/synthesis のロジックを使用） |
| 8 | グリッド拡張 | 拡張タブでスクラップを消費して行または列を追加する |
| 9 | 出撃 | 「出撃」ボタンで配置データを確定し BattleScene へ遷移する |

## 操作方法

| 入力 | アクション |
|------|-----------|
| タワーをドラッグ → グリッドへドロップ | タワーを配置する |
| グリッド上のタワーをドラッグ → グリッド外へドロップ | タワーを除去して一覧へ戻す |
| グリッド上のタワーをドラッグ → 別マスへドロップ | タワーを移動する |
| タワーにホバー | パラメータプレビューを表示する |
| 「合成」タブを選択 | 合成UIに切り替える |
| 素材A・B を選択 → 「合成する」ボタン | タワーを合成する |
| 「拡張」タブを選択 | グリッド拡張UIに切り替える |
| 「列を追加」または「行を追加」ボタン | スクラップを消費してグリッドを拡張する |
| 「出撃」ボタンをクリック | バトル開始 |

## ゲームプレイフロー

1. MapScene でバトルノードを選択すると PreparationScene へ遷移する
2. 「配置」タブが初期表示される。グリッドは前回バトル終了時の配置状態から開始する（初回は空グリッド）
3. プレイヤーは必要に応じてタブを切り替えて以下の操作を自由に行う
   - 配置タブ: タワーをグリッドへ配置・除去・移動する
   - 合成タブ: 所持タワー2つを選んで合成し、新しいタワーを得る
   - 拡張タブ: スクラップを消費してグリッドの行・列を増やす
4. 「出撃」ボタンを押すと配置データが RunManager に保存され BattleScene へ遷移する

## 各ゲーム状態の詳細

### 配置タブ / 待機中

- **表示内容**: 所持タワー一覧・戦車グリッド
- **プレイヤー操作**: タワーのドラッグ開始、ホバーでプレビュー表示、タブ切り替え
- **遷移条件**: タワーのドラッグ開始
- **遷移先**: ドラッグ中状態

### 配置タブ / ドラッグ中

- **表示内容**: タワーがカーソルに追従、グリッドの配置可否をハイライト（緑:可/赤:不可）
- **プレイヤー操作**: マウス移動でタワーを動かす、ドロップで配置・除去
- **遷移条件**: マウスボタンを離す
- **遷移先**: 待機中状態（配置成功 or キャンセル）

### 合成タブ

- **表示内容**: 素材A・B のドロップダウン、結果プレビュー（既知なら表示/未知なら???）、「合成する」ボタン、既知レシピ一覧
- **プレイヤー操作**: 素材を選択して「合成する」を押す
- **遷移条件**: タブ切り替え or 「出撃」
- **遷移先**: 他タブ or BattleScene

### 拡張タブ

- **表示内容**: 現在のグリッドサイズ、所持スクラップ数、「列を追加」「行を追加」ボタン（残高・上限に応じて有効/無効）
- **プレイヤー操作**: 拡張ボタンをクリックしてスクラップを消費しグリッドを拡張する
- **遷移条件**: タブ切り替え or 「出撃」
- **遷移先**: 他タブ or BattleScene

## グリッド仕様

| 項目 | 内容 |
|------|------|
| 初期サイズ | 3×3 マス |
| マスのサイズ | 64×64 px（UIスペース上） |
| 拡張単位 | 行または列を1つずつ追加（準備フェーズ・改修ノードの両方で拡張可） |
| 占有管理 | 各マスに TowerInstance を格納、null = 空き |

## タワーのサイズ形状仕様

| 表記 | 説明 | 例 |
|------|------|-----|
| 1×1 | 1マス | 全初期タワー |
| 1×2 | 横2マス | 将来の拡張タワー |
| 2×2 | 2×2マス | 将来の大型タワー |
| L字 | カスタム形状 | ScriptableObject で定義 |

## 勝利 / 敗北 / 終了条件

| 種別 | 条件 | 結果 |
|------|------|------|
| 遷移 | 「出撃」ボタンを押す | BattleScene へ遷移 |

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| マスが埋まっている場所へドロップ | ドロップをキャンセルし、タワーを元の位置へ戻す |
| タワーが所持枠からあふれている | スクロールで表示（一覧はスクロール対応） |
| グリッドにタワーが1つもない状態で出撃 | 警告ダイアログを表示せず許可（戦略的選択として有効） |
| 合成後にグリッド配置済みの素材タワーが消費された | グリッドから自動的に除去し、空きマスにする |
| グリッド拡張後にグリッドが最大サイズ（5×5）に達した | 「列を追加」「行を追加」ボタンをグレーアウト |
| スクラップ不足で拡張しようとした | ボタンをグレーアウト（押せない状態） |

## 未対応ケース

- グリッド上でのタワーの回転
- タワーの重ね置き・スタック
- プリセット保存（お気に入り配置の記憶）
