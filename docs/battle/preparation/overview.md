# 準備フェーズ 概要

## 目的・背景

バトル開始前にプレイヤーが戦車グリッドへタワーを配置するフェーズを実装する。タワーの選択・配置がバトル戦略の要であり、出撃前に合成やグリッド拡張も行えることで、ここでの判断がバトルの難易度・戦術を大きく左右する。

## スコープ

### 作るもの

- 戦車グリッドの表示・操作UI（初期3×3、拡張可）
- 所持タワー一覧の表示
- タワーのドラッグ＆ドロップによる配置・除去
- タワーサイズに応じた占有マス管理（重複配置の禁止）
- タワーのパラメータプレビュー（ホバー時）
- タワー合成（tower/synthesis のロジックを呼び出す）
- グリッド拡張（スクラップを消費して行・列を追加）
- 「出撃」ボタンによる BattleScene への遷移

### 作らないもの

- タワーの強化・売却（RefitScene / ShopScene で対応）
- 合成ロジックそのもの（tower/synthesis で実装し、ここから呼び出す）

## 制約

- グリッドサイズはデータとして保持し、UI はそれに動的に対応する
- タワーのサイズ形状はタワーデータ（ScriptableObject）で定義する
- 配置済みタワーの情報は RunManager へ渡し BattleScene で参照する

## 完了条件

- グリッドにタワーをドラッグ＆ドロップで配置できる
- 配置済みタワーをグリッドから外せる
- タワーサイズが2マス以上の場合も正しく占有判定される
- 合成タブで2つのタワーを選んで合成できる
- グリッド拡張タブでスクラップを消費してグリッドを拡張できる
- 「出撃」を押すと配置データが確定して BattleScene へ遷移する

```
|  所持タワー              |  戦車グリッド          |
|  ┌──────────┐           |  +---+---+---+         |
|  │ 蒸気砲   │ CT:2.0s  |  | A |   |   |         |
|  │ DMG:30   │           |  +---+---+---+         |
|  └──────────┘           |  | B | B |   |         |
|  ┌──────────┐           |  +---+---+---+         |
|  │ 機関銃   │ CT:0.3s  |  |   |   |   |         |
|  │ DMG:8    │           |  +---+---+---+         |
|  └──────────┘           |                        |
+--------------------------------------------------+
|  ▼ 合成タブ                                      |
|  [蒸気砲 ▼] + [機関銃 ▼] → [蒸気ガトリング]     |
|  [ 合成する ]                                    |
+--------------------------------------------------+
|  ▼ 拡張タブ                                      |
|  現在: 3×3  スクラップ: 120⚙                    |
|  [ 列を追加 +1  80⚙ ]  [ 行を追加 +1  80⚙ ]    |
+--------------------------------------------------+
```
