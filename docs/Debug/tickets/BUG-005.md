title: バトル中に敵が動かない
tags: [battle, enemy]

## 不具合内容

バトルフィールドに敵が配置されているが、一切動かない。

## 原因

`EnemyData.prefab` が未設定のため `FieldGenerator.PlaceEnemies()` の `if (data?.prefab == null) continue;` でスキップされ、敵が一体もスポーンされていなかった。

## 対処

- `FieldGenerator.SpawnEnemy()` を追加し、prefab が null の場合はコードでデフォルト GameObject（Rigidbody2D + CircleCollider2D + EnemyController + 白スプライト）を生成
- `EnemyController.Initialize(data)` を呼び出して AI を初期化
