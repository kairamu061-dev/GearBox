title: マップ導入スクロール後に一瞬暗転する
tags: [map, animation, ui]

---

## 不具合内容

新しいマップに入った際の導入スクロールアニメーション完了後、一瞬だけ画面が黒くなる。

## 原因

`IntroAnimation()` コルーチンの末尾で `SceneTransitionManager.FadeIn(0.4f)` を呼んでいたが、FadeIn は内部で Alpha を 1（黒）にセットしてから 0 にフェードするため、意図しない暗転が発生していた。シーン遷移時にすでにフェードイン済みであるため二重呼び出しになっていた。

## 対処

`MapSceneController.IntroAnimation()` 末尾の `FadeIn` 呼び出しを削除。
