# 強化パーツ（レリック）仕様

## 概要

所持しているだけで効果が発動するパッシブアイテム。スレイザスパイヤのレリックに相当する。バトル・準備フェーズなど場面を問わず、取得後は常時効果を発揮する。

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | パッシブ効果 | 所持中は常時効果が発動する（能動的な操作不要） |
| 2 | インベントリ管理 | RunManagerで所持中の強化パーツ一覧を管理する |
| 3 | 取得 | リザルト画面・ショップ・ハテナイベント等で入手できる |
| 4 | 効果適用 | 取得時にRunManagerへ登録し、即時効果を反映する |
| 5 | 一覧表示 | 準備フェーズや設定画面から所持一覧を確認できる（将来対応） |

## データ構造

| フィールド | 型 | 説明 |
|-----------|---|------|
| id | string | 強化パーツの一意ID |
| displayName | string | 表示名 |
| description | string | 効果の説明文 |
| effectType | RelicEffectType | 効果の種別 |
| effectValue | float | 効果の数値 |

## 入手経路

| 経路 | 条件 |
|------|------|
| 砦ドロップ | 砦破壊時のランダムドロップ |
| ショップ | ショップの在庫として出現（将来対応） |
| ハテナイベント | ランダムイベントの報酬（将来対応） |
| リザルト報酬 | バトル基本報酬の一部として出現（将来対応） |

## RunManagerへの追加

| フィールド | 型 | 初期値 | 説明 |
|-----------|---|--------|------|
| `RelicInventory` | List\<RelicData\> | 空 | 所持中の強化パーツ一覧 |

| メソッド | 説明 |
|---------|------|
| `AddRelic(RelicData r)` | 強化パーツを追加し、効果を即時適用する |

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 同一の強化パーツを複数取得 | 重複所持を許可する（効果が重複するかどうかはパーツ定義に依存） |

## 未対応ケース

- 強化パーツの売却・破棄
- 所持数上限の設定
- 効果の詳細一覧（効果種別は今後追加）
