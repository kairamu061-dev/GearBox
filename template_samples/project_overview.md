# プロジェクト概要

## ゲームコンセプト

重力を操作して障害物を避けながらゴールを目指す2Dパズルアクションゲーム。シンプルな操作で直感的に遊べるが、ステージが進むにつれて戦略性が求められる。

## コアループ

1. ステージ開始 → 重力の方向を切り替えながら移動
2. 障害物・ギミックを攻略してゴールに到達
3. クリアタイム・コレクションアイテム数でスコア評価
4. 次のステージへ進む or リトライ

## スコープ

### 実装するもの

- プレイヤー移動・重力切り替えメカニクス
- ステージ（全30面）
- タイトル・ゲーム・リザルト画面
- BGM・SE
- Steam実績（10種）

### 実装しないもの

- オンラインランキング
- ステージエディタ
- マルチプレイ

## ターゲットプラットフォーム

| 項目 | 内容 |
|------|------|
| プラットフォーム | Steam (PC) |
| OS | Windows 10以上 / macOS 12以上 |
| 想定プレイ時間 | 3〜5時間（全ステージクリア） |
| 価格帯 | 500〜800円 |

## 技術スタック概要

| 技術 | 用途 | 選定理由 |
|------|------|----------|
| Unity 2022 LTS | ゲームエンジン | 2D機能が充実・長期サポート |
| Steamworks SDK (Facepunch) | Steam連携・実績 | C#フレンドリーなラッパー |
| Unity Input System | 入力管理 | キーボード・ゲームパッド対応が容易 |
| DOTween | アニメーション | 軽量で直感的なTween制御 |

## 全体アーキテクチャ

```
シーン構成:
  TitleScene        ← タイトル・ステージ選択
  GameScene         ← ゲームプレイ本体
  ResultScene       ← クリア結果表示

主要 Manager（DontDestroyOnLoad）:
  GameManager       ← ゲーム状態管理
  AudioManager      ← BGM / SE再生
  SaveManager       ← セーブデータ管理
  SteamManager      ← Steam実績・統計
```

## 制約

- Unity 2022 LTS の標準 2D 機能のみで実装（3D物理エンジン不使用）
- ソロ開発のためアセットストア素材を活用する
- Steam リリースに必要な実績・クラウドセーブは必須対応

## UI デザイン方針

### コンセプト

ポップでカラフルなフラットデザイン。子どもから大人まで親しみやすいビジュアルを目指す。過度な装飾を避け、ゲームプレイの邪魔にならないシンプルなHUDを維持する。

### カラーパレット

| 用途 | カラーコード | 説明 |
|------|-------------|------|
| プライマリ | `#FF6B6B` | コーラルレッド。ボタン・強調に使用 |
| セカンダリ | `#4ECDC4` | ミントグリーン。アクセントに使用 |
| 背景 | `#2C3E50` | ダークネイビー。ゲーム背景 |
| UI テキスト | `#FFFFFF` | 白。HUD・UI文字 |
| 警告 / ダメージ | `#E74C3C` | 赤。HP減少・危険状態 |

### タイポグラフィ

- フォント: `Noto Sans JP`（日本語UI）、`Press Start 2P`（スコア・タイトル装飾）
- HUD: 16px bold
- タイトル: 32px

### コンポーネント方針

- **ボタン**: 角丸＋ドロップシャドウ、押下時にスケールアニメーション（DOTween）
- **HUD**: 画面四隅に配置。ゲームプレイ領域を最大限確保
- **ダイアログ**: 半透明黒背景のオーバーレイ形式
