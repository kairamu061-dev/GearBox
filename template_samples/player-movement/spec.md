# プレイヤー移動 仕様

## 機能一覧

| # | 機能名 | 説明 |
|---|--------|------|
| 1 | 左右移動 | 入力に応じてプレイヤーを左右に移動させる |
| 2 | 重力切り替え | ボタン入力で重力方向を上下に反転させる |
| 3 | 地面判定 | 地面・天井への着地を検出して移動を停止させる |
| 4 | 移動アニメーション | 状態に応じたアニメーションを再生する |

## 操作方法

| 入力 | アクション |
|------|-----------|
| A / ← | 左移動 |
| D / → | 右移動 |
| Space / ゲームパッドA | 重力切り替え |

## ゲームプレイフロー

1. ステージ開始：プレイヤーがスタート地点に出現する
2. プレイヤーがA/Dで左右に移動する
3. Spaceで重力方向が反転し、プレイヤーが逆方向に落下する
4. 新しい地面（天井）に着地して移動を再開する

## 各ゲーム状態の詳細

### Idle（待機）

- **表示内容**: 待機アニメーション再生
- **プレイヤー操作**: 左右移動入力 → Running 状態へ、重力切り替え → Falling 状態へ
- **遷移条件**: 移動入力なし・接地中
- **遷移先**: Running / Falling

### Running（移動中）

- **表示内容**: 走りアニメーション再生
- **プレイヤー操作**: 入力なし → Idle、重力切り替え → Falling
- **遷移条件**: 移動入力あり・接地中
- **遷移先**: Idle / Falling

### Falling（落下中）

- **表示内容**: 落下アニメーション再生
- **プレイヤー操作**: なし（重力に従い落下）
- **遷移条件**: 非接地
- **遷移先**: Idle（着地時）

## エラー / 異常ケース

| 条件 | 挙動 |
|------|------|
| 地面判定が複数フレームにわたり検出されない | フォールバックとして最後の接地位置にリセット |
| 重力切り替えが高速連打された | クールタイム（0.3秒）を設けて連続切り替えを防ぐ |

## 未対応ケース

- 斜面・坂道の移動は別タスクで対応
- 水中・特殊重力ゾーンは対象外
