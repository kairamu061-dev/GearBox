# プレイヤー移動 概要

## 目的・背景

ゲームのコアメカニクスである「重力切り替えを伴うプレイヤー移動」を実装する。プレイヤーは左右移動と重力方向の切り替えを組み合わせてステージを攻略する。

## スコープ

### 実装するもの

- 左右移動（キーボード・ゲームパッド対応）
- 重力切り替え（上下方向）
- 地面判定・着地処理
- 移動アニメーション（走る・待機・落下）

### 実装しないもの

- ダッシュ・ジャンプ（別機能エリアで対応）
- 壁つかまり

## 制約

- Unity Physics 2D（Rigidbody2D）を使用する
- ゲームパッドは Unity Input System で抽象化する

## 完了条件

- キーボード（A/D・矢印キー）で左右移動できる
- スペースキーまたはゲームパッドのボタンで重力が上下に切り替わる
- 地面に着地したときに正しく停止する
- 各状態に対応するアニメーションが再生される
