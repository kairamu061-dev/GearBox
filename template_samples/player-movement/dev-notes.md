# プレイヤー移動 開発メモ

## 実装上の判断

| 判断内容 | 理由 |
|----------|------|
| 物理移動に `Rigidbody2D.linearVelocity` を使用 | `MovePosition` では重力との競合が起きるため |
| 重力切り替えに `Physics2D.gravity` ではなく `Rigidbody2D.gravityScale` を使用 | 他オブジェクトへの影響を避けるためプレイヤー個別に管理 |
| 接地判定を `OverlapCircle` で実装 | Raycast より広めの判定で着地の取りこぼしを防ぐ |

## 発生した問題と対処

| 問題 | 対処 |
|------|------|
| 重力切り替え直後に接地判定が誤検出される | 切り替え後 1 フレームは接地判定を無効にする |
| 壁に接触したまま移動すると引っかかる | `PhysicsMaterial2D` の摩擦を 0 に設定して解消 |
